## 🔴 PE(TTM)计算错误发现（2026-03-29 16:34）

### 问题发现
在分析中际联合时，发现报告中PE(TTM)=20.39倍存在严重错误。

### 错误原因
```
错误计算:
PE = 股价 / 三季报累计EPS
   = 42.00 / 2.06 = 20.39倍 ❌

正确计算:
EPS(TTM) = 最新累计EPS + 去年年报EPS - 去年同期累计EPS
         = 2.06 + 1.48 - 1.12 = 2.42元
PE(TTM) = 42.00 / 2.42 = 17.36倍 ✅
```

### 核心教训
1. **PE(TTM)必须使用TTM EPS计算**
2. **TTM EPS公式**：最新累计EPS + 去年年报EPS - 去年同期累计EPS
3. **严禁用季度累计EPS替代TTM EPS**
4. **标注要准确**：如用季报EPS应标注PE(季报)而非PE(TTM)

## V3.3评分模型升级（晚间）

### 问题发现
东田微(301183)基本面评分仅18分，原因是stock_db中只有业绩预告数据，缺少正式财报完整数据。

### 设计讨论
- 业绩预告：发布早于正式报告，数据较少（仅有净利润、增速等），反映前瞻性预期
- 正式财报：数据完整、经过审计，但时效滞后

### 解决方案
**业绩预告与正式财报分离处理**：

| 数据类型 | 定位 | 用途 |
|----------|------|------|
| 正式财报 | 评分计算依据 | 基本面评分、估值计算 |
| 业绩预告 | 前瞻性洞察 | 趋势判断、预期分析 |

### V3.3更新内容

1. **基本面评分改进**
   - 只使用正式财报数据
   - 自动过滤业绩预告/快报

2. **业绩预告前瞻性分析**（独立模块）
   - 文件：`forecast_analysis.py`
   - 分析：趋势加速/放缓判断
   - 评估：预告确定性
   - 输出：前瞻性洞察

3. **报告结构调整**
   - 业绩预告分析结果展示在"基本信息-前瞻性洞察"
   - 不参与六维度综合评分

### 测试结果

**东田微(301183)**：
- V3.2: 基本面18分（数据缺失）
- V3.3: 基本面70.37分（使用正式财报）
- 前瞻性洞察：增速放缓，偏空

**中际旭创(300308)**：
- 基本面: 92.55分（正式财报）
- 前瞻性洞察：业绩加速向上，偏多
