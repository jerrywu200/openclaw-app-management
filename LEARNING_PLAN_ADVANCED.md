## 二、进阶学习方向

### 方向一：财务造假识别（专题深化）

**学习时间**：2-3天

**学习内容**：
```
1. 常见造假手段
   ├── 虚增收入
   │   └─ 关联交易、循环交易
   ├── 隐瞒负债
   │   └─ 表外融资、担保
   ├── 费用资本化
   │   └─ 研发费用、利息资本化
   └── 存货/应收账款操纵

2. 红旗信号识别
   ├── 应收账款异常
   ├── 存货异常
   ├── 现金流与利润背离
   ├── 频繁并购
   └── 关联交易频繁

3. 经典案例
   ├── 康美药业
   ├── 康得新
   ├── 瑞幸咖啡
   └── 其他A股案例

4. 实战练习
   └─ 分析可疑公司财报
```

**学习资源**：
- 《财务诡计》- 霍华德·施利特
- 夏草财务分析文章
- 证监会处罚案例

### 方向二：行业研究深化（重点行业覆盖）

**学习时间**：持续进行

**行业覆盖计划**：

| 优先级 | 行业 | 学习重点 | 预计时间 |
|--------|------|----------|----------|
| ⭐⭐⭐ | 半导体 | 国产替代、产业链 | 1周 |
| ⭐⭐⭐ | 新能源汽车 | 比亚迪、产业链 | 已开始 |

| ⭐ | 新能源 | 光伏、风电 | 2天 |

**每个行业研究框架**：
```
1. 行业概况
   └─ 规模、增速、驱动因素

2. 产业链分析
   └─ 上中下游、价值分布

3. 竞争格局
   └─ 波特五力、主要玩家

4. 商业模式
   └─ 盈利模式、护城河

5. 政策环境
   └─ 监管、扶持政策

6. 投资机会
   └─ 龙头公司、投资逻辑

7. 风险因素
   └─ 行业风险、公司风险
```

### 方向三：投资组合管理

**学习时间**：3-5天

**学习内容**：
```
1. 仓位管理
   ├── 凯利公式
   ├── 分散与集中
   ├── 动态调整
   └── 现金管理

2. 风险管理
   ├── 风险类型
   │   └─ 市场风险、行业风险、个股风险
   ├── 止损策略
   ├── 对冲策略
   └─ 黑天鹅应对

3. 投资组合构建
   ├── 核心持仓 vs 卫星持仓
   ├── 行业配置
   ├── 相关性管理
   └─ 再平衡策略

4. 心理管理
   ├── 认知偏差
   ├── 情绪控制
   └─ 投资纪律
```

**学习资源**：
- 《聪明的投资者》第8章（市场波动）
- 《穷查理宝典》
- 巴菲特致股东信

### 方向四：宏观经济分析

**学习时间**：1周

**学习内容**：
```
1. 宏观指标
   ├── GDP、CPI、PPI
   ├── 利率、汇率
   ├── 货币供应量
   └── 就业数据

2. 经济周期
   ├── 周期阶段识别
   ├── 行业周期与宏观周期
   └─ 周期投资策略

3. 政策分析
   ├── 货币政策
   │   └─ 降准、降息
   ├── 财政政策
   │   └─ 减税、支出
   └─ 产业政策

4. 国际环境
   ├── 美联储政策
   ├── 汇率影响
   └─ 地缘政治
```

**学习资源**：
- 《逃不开的经济周期》
- 瑞·达利欧《债务危机》
- 任泽平宏观分析
