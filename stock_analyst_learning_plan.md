## 📖 第一阶段：基础模块（1周）

### 模块一：财务报表分析

**学习目标**：掌握三表分析方法，识别财务红旗信号

**学习内容**：
```
Day 1: 资产负债表分析
├─ 核心等式：资产 = 负债 + 所有者权益
├─ 关键科目：应收账款、存货、商誉
├─ 财务健康指标：流动比率、速动比率、资产负债率
└─ 红旗信号：应收账款增速>营收增速、存货积压

Day 2: 利润表分析
├─ 核心流程：营收 → 毛利 → 营业利润 → 净利润
├─ 关键指标：毛利率、费用率、净利率
├─ 利润质量：净现比 = 经营现金流/净利润
└─ 红旗信号：毛利率异常波动、投资收益占比过高

Day 3: 现金流量表分析
├─ 三大现金流：经营/投资/筹资
├─ 核心判断：经营现金流持续为正
├─ 自由现金流 = 经营现金流 - 资本支出
└─ 红旗信号：靠融资+卖资产维持
```

**学习输出**：
- 能够独立分析一家公司的三表
- 识别至少5个财务红旗信号
- 完成3家公司的财务分析报告

**推荐练习**：
- 分析贵州茅台、宁德时代、比亚迪的财报

### 模块二：财务指标分析

**学习目标**：掌握四大类财务指标的计算和解读

**学习内容**：
```
Day 1: 盈利能力指标
├─ 毛利率 = (营收-成本)/营收
├─ 净利率 = 净利润/营收
├─ ROE = 净利润/净资产（>15%良好）
├─ ROA = 净利润/总资产
└─ ROIC = 税后经营利润/投入资本

Day 2: 偿债能力指标
├─ 流动比率 = 流动资产/流动负债（>1.5健康）
├─ 速动比率 = (流动资产-存货)/流动负债（>1健康）
├─ 资产负债率 = 总负债/总资产（<60%健康）
└─ 利息保障倍数 = EBIT/利息（>5很强）

Day 3: 运营效率指标
├─ 存货周转天数 = 365/(成本/平均存货)
├─ 应收周转天数 = 365/(营收/平均应收)
└─ 总资产周转率 = 营收/平均总资产

Day 4: 成长性指标
├─ 营收增长率（>30%高速）
├─ 净利润增长率
└─ 健康特征：利润增速 ≥ 营收增速
```

**学习输出**：
- 能够计算并解读15+个财务指标
- 完成1家公司的完整指标分析

### 模块四：估值方法

**学习目标**：掌握主流估值方法，能够进行合理估值

**学习内容**：
```
Day 1: 相对估值法
├─ PE（市盈率）= 股价/每股收益
│   ├─ <15低估，15-25合理，>40高估
│   └─ 适用：盈利稳定公司
├─ PEG = PE/盈利增长率
│   ├─ <1低估，=1合理，>2高估
│   └─ 适用：成长股
├─ PB（市净率）= 股价/每股净资产
│   ├─ <1破净，1-2合理，>3高估
│   └─ 适用：重资产、银行
└─ PS（市销率）= 市值/营收
    └─ 适用：亏损公司、高增长

Day 2: 绝对估值法
├─ DCF（现金流折现）
│   ├─ 企业价值 = Σ未来自由现金流折现
│   ├─ 关键参数：折现率(WACC)、永续增长率
│   └─ 注意事项：估值是区间不是精确数字
└─ 简化公式：合理PE ≈ 1/(要求回报率-永续增长率)

Day 3: 估值方法选择
├─ 消费/医药 → PE、PEG
├─ 银行/保险 → PB
├─ 科技/SaaS → PS、PEG
├─ 周期股 → PB（底部）
├─ 制造业 → EV/EBITDA
└─ 公用事业 → DCF

Day 4: 估值陷阱
├─ 周期股顶部低PE陷阱
├─ 亏损公司PE无意义
├─ 一次性收益扭曲PE
└─ 商誉减值风险
```

**学习输出**：
- 能够用3种以上方法对同一家公司估值
- 完成1家公司的完整估值报告

### 模块六：风险管理体系

**学习目标**：建立完整的风险管理框架

**学习内容**：
```
Day 1: 仓位管理
├─ 凯利公式：最优仓位 = (胜率×盈亏比-败率)/盈亏比
├─ 单股上限：10-15%
├─ 单行业上限：30%
└─ 现金保留：15-20%

Day 2: 止损策略
├─ 固定止损：单笔≤3%，日内≤5%
├─ 技术止损：跌破均线、支撑位
├─ 时间止损：买入后无盈利
└─ 执行纪律：触发即执行

Day 3: 止盈策略
├─ 固定止盈：3%/5%/8%分批
├─ 移动止盈：盈利后上移止损
├─ 技术止盈：压力位、背离
└─ 分批止盈：1/3、1/2、清仓

Day 4: 组合构建
├─ 分散化原理：15-20只股票分散80%非系统性风险
├─ 相关性矩阵：相关性<0.3分散效果好
└─ 动态再平衡：季度/阈值触发
```

**学习输出**：
- 制定个人交易纪律文档
- 完成1个模拟组合的构建
