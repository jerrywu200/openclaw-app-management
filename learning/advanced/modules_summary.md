## 模块五：技术分析进阶

### 一、均线系统

#### 1. 均线分类与含义

| 均线 | 周期 | 含义 |
|------|------|------|
| MA5 | 5日 | 短期趋势，超短线参考 |
| MA10 | 10日 | 短期趋势，短线支撑/压力 |
| MA20 | 20日 | 月线，中期趋势分界 |
| MA60 | 60日 | 季线，中期趋势确认 |
| MA120 | 120日 | 半年线，长期趋势 |
| MA250 | 250日 | 年线，牛熊分界线 |

#### 2. 均线排列判断

**多头排列**（看涨）:
```
价格 > MA5 > MA10 > MA20 > MA60
特征：短均线在上，长均线在下，发散向上
```

**空头排列**（看跌）:
```
价格 < MA5 < MA10 < MA20 < MA60
特征：短均线在下，长均线在上，发散向下
```

**缠绕状态**（震荡）:
```
均线交叉频繁，无明显方向
特征：趋势不明，等待突破
```

#### 3. 金叉与死叉

**金叉**（买入信号）:
- 短均线从下向上穿过长均线
- 有效性判断：
  - 长均线方向向上 → 有效
  - 长均线方向向下 → 可能假信号
  - 配合放量 → 可靠性增强

**死叉**（卖出信号）:
- 短均线从上向下穿过长均线
- 有效性判断：
  - 长均线方向向下 → 有效
  - 配合缩量 → 可靠性增强

#### 4. 均线实战技巧

```
支撑买入法：
- 多头排列中，股价回调至MA20/MA60获得支撑
- 成交量萎缩 → 买点

压力卖出法：
- 空头排列中，股价反弹至MA20/MA60遇到压力
- 成交量放大但滞涨 → 卖点

均线乖离率：
- 股价偏离MA20过远（>15%）→ 超买，注意回调
- 股价偏离MA20过远（<-15%）→ 超卖，可能反弹
```

### 二、形态识别

#### 1. 反转形态

**头肩顶**（看跌）:
```
    头
   /  \
左肩    右肩
  \    /
   颈线
```
- 特征：三个高点，中间最高
- 确认：跌破颈线
- 量度跌幅：头部到颈线的距离

**头肩底**（看涨）:
```
   颈线
  /    \
左肩    右肩
   \  /
    头
```
- 特征：三个低点，中间最低
- 确认：突破颈线
- 量度涨幅：头部到颈线的距离

**双顶（M头）**（看跌）:
```
  顶1    顶2
   \    /
    颈线
```
- 特征：两个相近高点
- 确认：跌破颈线

**双底（W底）**（看涨）:
```
    颈线
   /    \
底1      底2
```
- 特征：两个相近低点
- 确认：突破颈线

**圆弧顶/底**:
- 特征：缓慢形成，时间较长
- 含义：趋势缓慢转变

#### 2. 中继形态

**三角形整理**:

上升三角形（看涨）:
```
    _____
   /    /
  /    /
 /____/
```
- 特征：水平压力线，上升支撑线
- 突破方向：向上概率大

下降三角形（看跌）:
```
\      _____
 \    /
  \  /
   \/
```
- 特征：水平支撑线，下降压力线
- 突破方向：向下概率大

对称三角形:
```
    /\
   /  \
  /    \
 /      \
```
- 特征：收敛形态
- 突破方向：需结合趋势判断

**旗形整理**:
```
上升旗形：下降通道中的整理，突破后继续上涨
下降旗形：上升通道中的整理，突破后继续下跌
```

**矩形整理**:
```
_______
|     |
|     |
|_____|
```
- 特征：水平支撑和压力
- 突破方向：原趋势方向

#### 3. 缺口分析

| 缺口类型 | 位置 | 含义 |
|----------|------|------|
| 普通缺口 | 整理区 | 短期内回补，意义不大 |
| 突破缺口 | 整理末端 | 趋势启动信号，不易回补 |
| 中继缺口 | 趋势中段 | 趋势延续，测量目标位 |
| 衰竭缺口 | 趋势末端 | 趋势反转信号，会回补 |

### 三、量价关系深化

#### 1. 量价八法

| 编号 | 价格 | 成交量 | 含义 | 信号 |
|------|------|--------|------|------|
| 1 | 涨 | 放量 | 健康上涨 | 继续持有 |
| 2 | 涨 | 缩量 | 上涨乏力 | 谨慎 |
| 3 | 跌 | 放量 | 恐慌抛售 | 风险 |
| 4 | 跌 | 缩量 | 正常回调 | 观察支撑 |
| 5 | 平 | 放量 | 多空分歧 | 变盘前兆 |
| 6 | 平 | 缩量 | 观望气氛 | 继续整理 |
| 7 | 量增价不涨 | 放量滞涨 | 主力出货 | 卖出 |
| 8 | 量缩价不跌 | 缩量抗跌 | 主力护盘 | 关注 |

#### 2. 成交量验证原则

```
价升量增 → 趋势健康，可持有
价升量缩 → 动能不足，谨慎
价跌量增 → 抛压加重，规避
价跌量缩 → 洗盘或底部，观察
天量天价 → 高位放量滞涨，见顶信号
地量地价 → 底部极度萎缩，见底信号
```

#### 3. 筹码分布

**筹码集中度判断**:
- 筹码峰在低位 → 成本集中，安全
- 筹码峰在高位 → 套牢盘多，压力大
- 筹码发散 → 成本分散，趋势不明

**筹码移动**:
- 低位筹码不移动 → 主力锁仓，看好
- 低位筹码上移 → 主力出货，风险
- 高位筹码下移 → 割肉盘出局，底部形成

## 模块六：风险管理体系

### 一、仓位管理

#### 1. 仓位管理原则

**核心原则**: 不要把鸡蛋放在一个篮子里

| 风险偏好 | 单只股票上限 | 行业上限 | 现金比例 |
|----------|--------------|----------|----------|
| 保守型 | 10% | 20% | 30%+ |
| 稳健型 | 15% | 30% | 20% |
| 激进型 | 25% | 40% | 10% |

#### 2. 金字塔建仓法

```
第一次：40%仓位（底仓）
    ↓
第二次：30%仓位（确认趋势加仓）
    ↓
第三次：20%仓位（趋势延续加仓）
    ↓
第四次：10%仓位（最后加仓）

特点：越涨越买，仓位成本递增，但均价可控
```

#### 3. 倒金字塔减仓法

```
第四次加仓部分 → 先卖出（最高价）
    ↓
第三次加仓部分 → 次卖出
    ↓
第二次加仓部分 → 再卖出
    ↓
第一次底仓部分 → 最后卖出（利润最大）

特点：先卖高成本的，保护利润
```

#### 4. 凯利公式

```
最优仓位比例 = (胜率 × 盈亏比 - 败率) / 盈亏比

示例：
胜率 = 60%，盈亏比 = 2:1
最优仓位 = (0.6 × 2 - 0.4) / 2 = 40%

保守使用：实际仓位 = 凯利仓位 × 50%
```

### 二、止损策略

#### 1. 止损方法

| 方法 | 设置方式 | 适用场景 |
|------|----------|----------|
| 固定比例止损 | 亏损8-10%止损 | 通用 |
| 技术止损 | 跌破支撑位/均线 | 技术派 |
| 时间止损 | 持有N天不涨则退出 | 短线 |
| 波动止损 | ATR的2-3倍 | 波动大的股票 |

#### 2. 止损执行原则

```
1. 买入前就设好止损位，买入后严格执行
2. 止损不能随意移动（只能上移保护利润）
3. 止损触发后立即执行，不犹豫
4. 止损后不急于回补，等待新信号
```

#### 3. 止盈策略

| 方法 | 设置方式 | 适用场景 |
|------|----------|----------|
| 固定比例止盈 | 盈利20-30%止盈 | 短线 |
| 移动止盈 | 最高价回撤10%止盈 | 中线 |
| 分批止盈 | 20%卖1/3，40%卖1/3，60%卖剩余 | 长线 |
| 目标价止盈 | 达到估值目标卖出 | 价值投资 |

#### 4. 盈亏比概念

```
盈亏比 = 预期盈利 / 预期亏损

示例：
买入价100元，止盈120元，止损92元
盈亏比 = (120-100) / (100-92) = 20 / 8 = 2.5

要求：
- 盈亏比 > 2 才值得交易
- 盈亏比 < 1.5 不建议交易
```

### 三、组合构建

#### 1. 资产配置

**股债平衡策略**:
```
保守型：股票30% + 债券70%
稳健型：股票50% + 债券50%
激进型：股票70% + 债券30%
```

**年龄法则**:
```
股票比例 = 100 - 年龄
债券比例 = 年龄

示例：30岁 → 股票70%，债券30%
```

#### 2. 行业分散

```
核心原则：单一行业占比不超过30%

推荐配置：
- 消费（白酒、医药）：20-30%
- 科技（半导体、新能源）：20-30%
- 金融（银行、保险）：15-20%
- 周期（有色、化工）：10-15%
- 其他：10-20%
```

#### 3. 风险评估指标

**最大回撤**:
```
最大回撤 = (峰值 - 谷值) / 峰值

评估：
- <10%：低风险
- 10-20%：中等风险
- 20-30%：较高风险
- >30%：高风险
```

**夏普比率**:
```
夏普比率 = (收益率 - 无风险收益率) / 波动率

评估：
- >1：良好
- >2：优秀
- >3：卓越
```

## 模块七：行业深度研究

### 一、行业生命周期详解

#### 1. 生命周期四阶段

| 阶段 | 渗透率 | 增速 | 特征 | 投资策略 |
|------|--------|------|------|----------|
| 导入期 | <10% | 不稳定 | 技术不成熟，需求小 | 观察，少量布局龙头 |
| **成长期** | **10-50%** | **>20%** | **需求爆发，竞争加剧** | **重点配置** |
| 成熟期 | 50-80% | 5-15% | 竞争格局稳定，龙头盈利 | 配置龙头 |
| 衰退期 | >80% | <0% | 需求萎缩，产能过剩 | 回避 |

#### 2. 判断渗透率的方法

```
渗透率 = 某产品销量 / 潜在市场总量

数据来源：
- 行业协会数据
- 券商研报
- 公司公告
- 政府统计
```

### 二、产业链分析方法

#### 1. 产业链梳理框架

```
上游（原材料）
    ↓ 供应关系
中游（制造/加工）
    ↓ 销售关系
下游（应用/终端）

分析要点：
1. 各环节的集中度
2. 议价能力强弱
3. 利润分配格局
4. 瓶颈环节在哪
```

#### 2. 议价能力判断

**上游议价能力**:
- 原材料稀缺性
- 供应商集中度
- 替代品可得性

**下游议价能力**:
- 客户集中度
- 产品差异化程度
- 转换成本高低

#### 3. 产业链案例：新能源汽车

```
上游：
- 锂矿：天齐锂业、赣锋锂业（资源稀缺，议价强）
- 钴镍：华友钴业（周期性强）

中游：
- 正极材料：容百科技、当升科技
- 负极材料：贝特瑞、璞泰来
- 电解液：天赐材料
- 隔膜：恩捷股份
- 电池：宁德时代（龙头，议价强）

下游：
- 整车：比亚迪、特斯拉
- 终端：消费者（议价弱）

利润分配：
- 上游资源 > 中游电池 > 整车 > 材料
```

### 三、竞争格局分析

#### 1. 市场集中度指标

**CR4（前四名集中度）**:
```
CR4 = 前四名企业市场份额之和

判断：
- CR4 < 30%：竞争型市场
- CR4 30-50%：低集中寡占型
- CR4 50-70%：中集中寡占型
- CR4 > 70%：高集中寡占型
```

**赫芬达尔指数（HHI）**:
```
HHI = Σ(企业市场份额)²

判断：
- HHI < 1000：竞争型
- HHI 1000-1800：中度集中
- HHI > 1800：高度集中
```

#### 2. 竞争格局演变

```
早期：群雄逐鹿，玩家众多
    ↓
中期：洗牌整合，龙头崛起
    ↓
后期：双寡头/多寡头，格局稳定

投资机会：中期布局龙头，享受集中度提升红利
```

### 四、政策影响分析

#### 1. 政策类型

| 政策类型 | 影响 | 案例 |
|----------|------|------|

```

   - 国务院文件
   - 部委通知
   - 行业协会公告

2. 判断政策力度：
   - 鼓励类：利好
   - 限制类：利空
   - 禁止类：重大利空

3. 评估政策持续时间：
   - 短期政策：影响1-2年
   - 中期政策：影响3-5年
   - 长期政策：影响5-10年
```

## 学习心得

- 均线是趋势的朋友，顺趋势操作

- 选对赛道比选对公司更重要
- 渗透率是判断行业阶段的关键
- 产业链分析找到最有价值的环节
