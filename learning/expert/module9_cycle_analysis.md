## 一、核心知识掌握

### 1. 周期股分析方法

**PB估值框架**：

| PB位置 | 判断 | 操作 |
|--------|------|------|
| PB < 1 | 历史底部 | ✅ 买入机会 |
| PB 1-2 | 估值偏低 | ✅ 可布局 |
| PB 2-3 | 估值合理 | ⚠️ 持有 |
| PB 3-5 | 估值偏高 | ⚠️ 考虑减仓 |
| PB > 5 | 历史高位 | 🔴 卖出信号 |

**三重周期**：
- 产能周期（3-5年）：最长，决定大方向
- 价格周期（1-3年）：中周期，决定盈利波动
- 库存周期（3-6个月）：短周期，影响短期波动

**底部信号**：
1. ⭐⭐⭐⭐⭐ 全行业亏损
2. ⭐⭐⭐⭐ PB破净
3. ⭐⭐⭐⭐ 龙头减产
4. ⭐⭐⭐ 并购重组
5. ⭐⭐⭐ 跨界退出

**顶部信号**：
1. ⭐⭐⭐⭐⭐ 疯狂扩产
2. ⭐⭐⭐⭐⭐ 跨界进入
3. ⭐⭐⭐⭐ PB历史高位
4. ⭐⭐⭐⭐ 业绩高增长（需警惕）

### 2. 成长股生命周期

**四阶段模型**：

| 阶段 | 营收增速 | 利润率 | 竞争格局 | 投资机会 |
|------|----------|--------|----------|----------|
| 导入期 | 低 | 负/低 | 少 | 高风险 |
| **成长期** | **>30%** | **改善** | **多** | **⭐⭐⭐⭐⭐** |
| 成熟期 | 5-15% | 稳定 | 寡头 | ⭐⭐⭐ |
| 衰退期 | 低/负 | 下滑 | 存活少 | 回避 |

**渗透率与投资机会**：

| 渗透率 | 阶段 | 投资策略 |
|--------|------|----------|
| <5% | 导入期 | 观察 |
| 5-10% | 导入期后期 | 开始关注 |
| **10-50%** | **成长期** | **✅ 最佳投资期** |
| 50-80% | 成熟期 | 选龙头 |
| >80% | 衰退期 | 回避 |

### 3. 行业景气度追踪

**量价关系矩阵**：

| 价格 | 销量 | 景气度 |
|------|------|--------|
| ↑ | ↑ | ⭐⭐⭐⭐⭐ 最高 |
| ↑ | → | ⭐⭐⭐⭐ 高 |
| → | ↑ | ⭐⭐⭐⭐ 高 |
| ↑ | ↓ | ⭐⭐⭐ 中等 |
| ↓ | ↑ | ⭐⭐⭐ 中等 |
| ↓ | ↓ | ⭐ 最低 |

**产能利用率判断**：
- >90%：满产，供不应求
- 80-90%：景气度高
- 70-80%：景气度中等
- <70%：产能过剩

### 4. 周期反转信号

**底部反转信号组合**：
- 价格跌破成本线
- 全行业亏损
- 龙头企业减产
- 库存去化加速
- PB<1
- 政策扶持

**判断标准**：3个以上信号同时出现 → 底部反转概率高

**顶部反转信号组合**：
- 疯狂扩产
- 跨界进入
- PB历史高位
- 业绩顶峰
- 价格远超成本

## 四、投资决策框架

### 周期股决策树

```
PB历史分位判断
  ├─ PB<20% → 底部区域 → 检查底部信号
  │   └─ 3个以上信号 → 建仓
  │   └─ 信号不足 → 等待
  ├─ PB 20-80% → 中部区域 → 持有
  └─ PB>80% → 顶部区域 → 检查顶部信号
      └─ 3个以上信号 → 减仓
      └─ 信号不足 → 观察
```

### 成长股决策树

```
生命周期判断
  ├─ 导入期（渗透率<10%）→ 观察
  ├─ 成长期（渗透率10-50%）→ 最佳投资期
  │   ├─ 景气度高 → 积极布局
  │   └─ 景气度中等 → 逢低布局
  ├─ 成熟期（渗透率50-80%）→ 选龙头
  └─ 衰退期（渗透率>80%）→ 回避
```
