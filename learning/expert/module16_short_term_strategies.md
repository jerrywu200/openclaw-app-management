## 一、短线买入策略

### 1.1 开盘强势买入

**适用条件**：
```
市场环境：
- 大盘平稳或上涨
- 板块热度高
- 个股有利好

技术条件：
- 高开 > 2%
- 量比 > 2
- 委比 > 50%
```

**操作流程**：
```
9:25 竞价结束：
□ 观察高开幅度
□ 检查量比
□ 确认板块热度

9:30 开盘：
□ 观察前5分钟走势
□ 是否站稳均价线

9:35 决策：
□ 如站稳均价线 → 买入
□ 如快速回落 → 放弃

止损：跌破开盘价-2%
```

**示例**：
```
某股昨收100元
竞价：高开3%至103元
量比：2.5
开盘5分钟：价格在102.5-103.5震荡，均价线在103
判断：强势，可买入
操作：103元附近买入
止损：101元（开盘价-2%）
```

### 1.2 回调买入

**适用条件**：
```
趋势：上升趋势中
位置：回调至支撑位
量能：量能萎缩
```

**支撑位选择**：
```
优先级：
1. MA5/MA10均线
2. 分时均价线（黄线）
3. 前日收盘价
4. 布林中轨
5. 前低点
```

**操作流程**：
```
观察阶段：
□ 价格回调
□ 量能萎缩
□ 接近支撑位

确认阶段：
□ 支撑位企稳
□ 出现反弹迹象
□ 量能开始放大

执行阶段：
□ 支撑位附近买入
□ 止损设支撑位下方2%

止盈：
□ 反弹至前高
□ 冲高回落
```

**示例**：
```
某股上升趋势，MA5在50元
回调过程：
- 价格从52元回调
- 量能逐步萎缩
- 接近MA5（50元）

买入信号：
- 价格触及50.2元后反弹
- 量能开始放大
- KDJ超卖区金叉

操作：50.5元买入
止损：49元（MA5下方2%）
止盈：52-53元
```

### 1.3 突破买入

**突破类型**：
```
1. 突破前高
   - 突破近期高点
   - 量能放大确认
   - 回踩不破前高

2. 突破平台
   - 突破横盘整理区间
   - 量能放大
   - 不回落至平台内

3. 突破均线
   - 突破重要均线（MA20/MA60）
   - 量能放大
   - 站稳确认
```

**操作流程**：
```
突破观察：
□ 价格接近关键位置
□ 量能开始放大
□ 观察是否有效突破

确认标准：
□ 突破幅度 > 3%
□ 量比 > 1.5
□ 收盘站稳

执行：
□ 突破确认后买入
□ 或回踩确认买入
□ 止损设突破位下方3%
```

**假突破识别**：
```
特征：
- 突破后快速回落
- 量能萎缩
- 收盘跌破关键位

处理：
- 突破后观察1-2根K线
- 如回落至关键位以下 → 放弃
- 假突破后常有大跌
```

### 1.4 尾盘潜伏

**适用条件**：
```
时间：14:30-15:00
特征：尾盘异动
目的：博弈次日高开
```

**尾盘信号**：
```
买入信号：
1. 尾盘拉升
   - 14:40后开始上涨
   - 量能放大
   - 突破均价线

2. 尾盘站稳
   - 全天强势
   - 尾盘不跳水
   - 维持高位

3. 尾盘吸筹
   - 尾盘大买单
   - 价格稳步上升
   - 主力行为
```

**操作要点**：
```
时间选择：
- 14:40开始关注
- 14:50确认方向
- 14:55执行买入

注意事项：
- 避免追高（涨幅>5%不追）
- 确认不是诱多
- 次日开盘注意止盈
```

## 二、短线卖出策略

### 2.1 冲高回落卖出

**识别信号**：
```
特征：
- 价格冲高后回落
- 跌破分时均价线
- 量能萎缩或放大
- 委比转负

卖出时机：
- 跌破均价线后反弹无力 → 卖出
- 反弹至均价线附近 → 减仓
```

**操作流程**：
```
观察：
□ 价格冲高
□ 开始回落
□ 接近均价线

确认：
□ 跌破均价线
□ 反弹无力
□ 委比转负

执行：
□ 反弹至均价线附近卖出
□ 或跌破均价线立即卖出
```

### 2.2 技术死叉卖出

**均线死叉**：
```
MA5死叉MA10：
- 价格在高位
- MA5下穿MA10
- 量能放大
→ 卖出信号

注意：
- 死叉位置越高越危险
- 确认跌破MA5后再操作
```

**MACD死叉**：
```
情况一：高位死叉
- MACD在零轴上方
- DIF下穿DEA
- 红柱转绿
→ 卖出

情况二：顶背离死叉
- 价格创新高
- MACD不创新高
- 死叉确认
→ 强烈卖出
```

**KDJ死叉**：
```
超买区死叉（K,D>80）：
- 最强卖出信号
- 准确率高
→ 立即卖出

高位死叉（K,D>60）：
- 卖出信号
- 注意回调
→ 减仓或卖出
```

### 2.3 顶背离卖出

**识别顶背离**：
```
价格：创新高
指标：不创新高

常见组合：
1. 价格新高 + MACD不新高 → 顶背离
2. 价格新高 + RSI不新高 → 顶背离
3. 价格新高 + KDJ不新高 → 顶背离
```

**操作策略**：
```
发现背离：
- 第一次背离 → 警惕，准备减仓
- 第二次背离 → 强烈卖出信号

配合确认：
- 量能萎缩
- 涨幅较大
- 技术指标走弱

执行：
- 背离确认后减仓
- 死叉确认后清仓
```

### 2.4 止盈卖出

**固定止盈**：
```
盈利目标：
- 盈利3% → 止损移至成本价
- 盈利5% → 止损移至+2%
- 盈利8% → 止盈一半
- 盈利10% → 止盈大部分
```

**分批止盈**：
```
策略：
第一目标（+5%）：卖出1/3
第二目标（+8%）：卖出1/2
第三目标（+12%）：清仓

剩余部分：
- 移动止损跟随
- 让利润奔跑
- 防止卖飞
```

**技术止盈**：
```
信号：
- 触及压力位
- 量能异常放大
- 技术指标超买

操作：
- 冲高回落 → 卖出
- 触及目标价 → 卖出
- 指标死叉 → 卖出
```

## 三、T+0策略（日内回转）

### 3.1 T+0前提条件

```
基本条件：
□ 已有底仓
□ 持仓数量充足
□ 有现金可用

市场条件：
□ 震荡行情（非单边）
□ 日内波动>3%
□ 有明确支撑压力
```

### 3.2 正向T（先买后卖）

**适用场景**：
```
市场：低开或盘中回调
位置：支撑位附近
操作：先买入后卖出
```

**操作流程**：
```
开盘观察：
□ 低开或回调
□ 接近支撑位
□ 量能萎缩

第一笔买入：
□ 支撑位附近买入
□ 数量不超过底仓
□ 止损设支撑位下方

等待反弹：
□ 反弹至压力位
□ 量能确认
□ 技术信号配合

第二笔卖出：
□ 卖出相同数量
□ 完成T+0
□ 锁定差价

示例：
底仓：1000股，成本50元
操作：
- 48元买入1000股
- 50元卖出1000股
- 收益：(50-48)×1000 = 2000元
```

### 3.3 反向T（先卖后买）

**适用场景**：
```
市场：高开或盘中冲高
位置：压力位附近
操作：先卖出后买回
```

**操作流程**：
```
开盘观察：
□ 高开或冲高
□ 接近压力位
□ 量能放大

第一笔卖出：
□ 压力位附近卖出
□ 数量不超过底仓
□ 止损设压力位上方

等待回调：
□ 回调至支撑位
□ 量能萎缩
□ 技术信号配合

第二笔买回：
□ 买回相同数量
□ 完成T+0
□ 锁定差价

示例：
底仓：1000股，成本50元
操作：
- 52元卖出1000股
- 50元买回1000股
- 收益：(52-50)×1000 = 2000元
```

### 3.4 T+0注意事项

```
时间控制：
- 尽量在10:30前完成
- 避免尾盘操作
- 留足时间应对

利润目标：
- 单次T利润1-3%
- 不贪心
- 积少成多

止损纪律：
- 判断失误立即止损
- 不拖延
- 保护底仓

仓位控制：
- T的仓位不超过底仓
- 保留底仓不动
- 滚动操作
```

## 四、短线策略组合

### 4.1 早盘策略（9:30-10:30）

```
高开高走策略：
1. 竞价观察：高开>2%，量比>1.5
2. 开盘5分钟：站稳均价线
3. 操作：回调至均价线买入
4. 止损：跌破开盘价-2%
5. 止盈：冲高回落或涨幅>5%

低开高走策略：
1. 竞价观察：低开<2%
2. 开盘观察：快速翻红
3. 操作：翻红后回调买入
4. 止损：跌破分时新低
5. 止盈：压力位或涨幅>3%

低开低走策略：
1. 观望为主
2. 等待企稳
3. 不急于抄底
```

### 4.2 盘中策略（10:30-14:30）

```
震荡策略：
1. 确认震荡区间
2. 下沿买入，上沿卖出
3. 设置止损
4. 不追涨杀跌

趋势策略：
1. 确认趋势方向
2. 顺势操作
3. 回调买入
4. 移动止损

变盘策略：
1. 观察变盘信号
2. 突破确认
3. 顺势跟进
4. 止损严格
```

### 4.3 尾盘策略（14:30-15:00）

```
尾盘买入：
- 条件：全天强势，尾盘站稳
- 操作：14:50附近买入
- 目的：博弈次日高开
- 止损：次日开盘-2%

尾盘卖出：
- 条件：尾盘跳水
- 操作：反弹卖出
- 目的：避免次日低开
- 替代：次日开盘卖出

尾盘观察：
- 确认趋势
- 不盲目操作
- 准备次日策略
```

## 五、短线交易纪律

### 5.1 操作纪律

```
入场纪律：
□ 信号确认后才操作
□ 不凭感觉交易
□ 严格按计划执行

持仓纪律：
□ 盯紧止损
□ 不贪婪
□ 按计划止盈

离场纪律：
□ 止损触发立即执行
□ 不抱侥幸心理
□ 保护本金第一
```

### 5.2 资金管理

```
单笔控制：
- 单笔投入 ≤ 总资金10%
- 单笔止损 ≤ 3%
- 单笔止盈 ≥ 5%

日内控制：
- 日内总亏损 ≤ 5%
- 连续2次亏损 → 停止操作
- 日内盈利 → 可追加操作

仓位控制：
- 震荡市：轻仓操作
- 趋势市：正常仓位
- 不确定：观望
```

### 5.3 心态管理

```
正确心态：
- 不贪心
- 不恐惧
- 严格执行
- 接受亏损

错误心态：
- 追涨杀跌
- 报复性交易
- 拖延止损
- 过度自信

调节方法：
- 设定每日交易次数上限
- 亏损后休息
- 保持冷静
```
