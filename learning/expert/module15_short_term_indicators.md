## 一、均线系统（短线版）

### 1.1 短线均线参数

| 均线 | 周期 | 短线含义 | 用途 |
|------|------|----------|------|
| MA5 | 5日 | 超短线支撑/压力 | 日内短线参考 |
| MA10 | 10日 | 短线趋势线 | 短线多空分界 |
| MA20 | 20日 | 月线，波段趋势 | 波段操作参考 |
| MA60 | 60日 | 季线，中期趋势 | 中期趋势确认 |

### 1.2 均线多头/空头排列

**多头排列（看涨）**：
```
价格
  ↑  最高
MA5
MA10
MA20
MA60
  ↓  最低

特征：
- 短期均线在上，长期均线在下
- 均线发散向上
- 趋势向上

操作：逢低买入，持有为主
```

**空头排列（看跌）**：
```
MA60
  ↓  最高
MA20
MA10
MA5
价格
  ↑  最低

特征：
- 长期均线在上，短期均线在下
- 均线发散向下
- 趋势向下

操作：逢高卖出，观望为主
```

### 1.3 金叉与死叉

**金叉（买入信号）**：
```
         ╱
        ╱
MA5  ╱     ← MA5上穿MA10
    ╱   ╱
   ╱   ╱  MA10
  ╱   ╱

判断标准：
1. MA5从下方向上穿越MA10
2. 金叉位置越低越好
3. 量能配合放大
4. 趋势向上时金叉更可靠

操作：短线买入
```

**死叉（卖出信号）**：
```
MA5
  ╲   ╱  MA10
   ╲ ╱
    ╳    ← MA5下穿MA10
   ╱ ╲
  ╱   ╲

判断标准：
1. MA5从上方向下穿越MA10
2. 死叉位置越高越危险
3. 量能放大
4. 趋势向下时死叉更危险

操作：短线卖出
```

### 1.4 均线支撑与压力

```
支撑作用：
- 价格回调至MA5/MA10获得支撑
- 均线向上
- 量能萎缩
→ 买入机会

压力作用：
- 价格反弹至MA5/MA10遇到阻力
- 均线向下
- 量能萎缩
→ 卖出机会

判断技巧：
- 均线角度越陡，支撑/压力越强
- 多条均线粘合处，支撑/压力更强
```

## 二、MACD短线应用

### 2.1 MACD基础

```
MACD构成：
├─ DIF线（快线）：12日EMA - 26日EMA
├─ DEA线（慢线）：DIF的9日EMA
└─ MACD柱：(DIF - DEA) × 2

参数：默认(12, 26, 9)

短线建议：
- 日线MACD滞后，适合波段
- 60分钟MACD适合短线
- 30分钟MACD适合超短线
```

### 2.2 MACD信号

**金叉买入**：
```
DIF           ╱
    ╲       ╱
     ╲     ╱
      ╲   ╱
       ╲ ╱  DEA
        ╳
       ╱ ╲

信号强度：
1. 零轴下方金叉 → 弱反弹，谨慎
2. 零轴附近金叉 → 转强信号，关注
3. 零轴上方金叉 → 强势，可追

配合条件：
- 量能放大
- 趋势向上
- 红柱放大
```

**死叉卖出**：
```
DIF
  ╲
    ╲
      ╲ ╱  DEA
        ╳
       ╱ ╲
      ╱   ╲
     ╱     ╲

信号强度：
1. 零轴上方死叉 → 调整开始，减仓
2. 零轴附近死叉 → 转弱信号，警惕
3. 零轴下方死叉 → 弱势加速，观望

配合条件：
- 量能放大
- 趋势向下
- 绿柱放大
```

### 2.3 MACD背离

**顶背离（卖出信号）**：
```
价格:   ↗ ↗ ↗  (创新高)
MACD:   ↗ ↘ ↗  (不创新高)

含义：价格上涨，但MACD不创新高
      上涨动能减弱，可能见顶

确认条件：
1. DIF出现明显高点
2. 第二高点低于第一高点
3. 价格对应创新高

操作：逢高减仓或卖出
```

**底背离（买入信号）**：
```
价格:   ↘ ↘ ↘  (创新低)
MACD:   ↘ ↗ ↘  (不创新低)

含义：价格下跌，但MACD不创新低
      下跌动能减弱，可能见底

确认条件：
1. DIF出现明显低点
2. 第二低点高于第一低点
3. 价格对应创新低

操作：逢低布局
```

## 三、KDJ指标

### 3.1 KDJ基础

```
KDJ构成：
├─ K线：快速指标
├─ D线：慢速指标
└─ J线：超买超卖敏感指标

参数：默认(9, 3, 3)

计算逻辑：
1. RSV = (收盘-最低)/(最高-最低) × 100
2. K = RSV的3日移动平均
3. D = K的3日移动平均
4. J = 3K - 2D
```

### 3.2 KDJ区间

```
区间划分：
├─ K/D > 80  → 超买区
├─ 20 < K/D < 80 → 正常区
└─ K/D < 20  → 超卖区

J线极端值：
├─ J > 100  → 极度超买
└─ J < 0    → 极度超卖
```

### 3.3 KDJ信号

**超买区死叉（强烈卖出）**：
```
K, D > 80
     K  ╲
         ╲
          ╲  D
           ╲╲
            ╳  ← 死叉
           ╱╱
          ╱

信号强度：⭐⭐⭐⭐⭐
- 超买区死叉是最强卖出信号
- 准确率较高
- 及时止盈

操作：立即卖出
```

**超卖区金叉（强烈买入）**：
```
K, D < 20
           ╱╱
          ╱
         ╱  D
     K  ╱
        ╱
       ╳  ← 金叉

信号强度：⭐⭐⭐⭐⭐
- 超卖区金叉是最强买入信号
- 准确率较高
- 抄底良机

操作：大胆买入
```

### 3.4 KDJ使用要点

```
注意事项：

1. 钝化问题
   - 强势股KDJ长期高位钝化 → 不急于卖出
   - 弱势股KDJ长期低位钝化 → 不急于买入
   - 解决：结合趋势使用

2. 顺势使用
   - 上升趋势：超卖金叉更可靠
   - 下降趋势：超买卖死叉更可靠

3. 配合量能
   - 金叉+放量 → 更可靠
   - 死叉+放量 → 更危险

4. 多周期确认
   - 日线KDJ金叉 + 60分钟金叉 → 更可靠
```

## 四、RSI指标

### 4.1 RSI基础

```
RSI（相对强弱指标）
参数：常用14日或6日

公式：
RSI = 100 - 100/(1+RS)
RS = 平均上涨幅度/平均下跌幅度
```

### 4.2 RSI区间

```
区间划分：
├─ RSI > 80  → 严重超买，强烈卖出信号
├─ 70 < RSI < 80 → 超买区，注意回调
├─ 50 < RSI < 70 → 强势区，多头占优
├─ 30 < RSI < 50 → 弱势区，空头占优
├─ 20 < RSI < 30 → 超卖区，关注反弹
└─ RSI < 20  → 严重超卖，强烈买入信号

50线是多空分界线：
- RSI > 50：多头市场
- RSI < 50：空头市场
```

### 4.3 RSI信号

**超买转头向下**：
```
RSI
  │    ╱╲
  │   ╱  ╲
80├──╱────╲─── 超买线
  │ ╱      ╲
  │╱        ╲
  └─────────────

信号：
- RSI从超买区转头向下
- 配合价格下跌
→ 卖出信号
```

**超卖转头向上**：
```
RSI
  │╲        ╱
  │ ╲      ╱
30├──╲────╱─── 超卖线
  │   ╲  ╱
  │    ╲╱
  └─────────────

信号：
- RSI从超卖区转头向上
- 配合价格上涨
→ 买入信号
```

### 4.4 RSI背离

**顶背离**：
```
价格: ↗ ↗ ↗  (创新高)
RSI:  ↗ ↘ ↗  (不创新高)

含义：上涨动能衰竭
操作：卖出
```

**底背离**：
```
价格: ↘ ↘ ↘  (创新低)
RSI:  ↘ ↗ ↘  (不创新低)

含义：下跌动能衰竭
操作：买入
```

## 五、布林带

### 5.1 布林带基础

```
布林带构成：
├─ 上轨 = 中轨 + 2×标准差
├─ 中轨 = 20日移动平均线
└─ 下轨 = 中轨 - 2×标准差

参数：默认(20, 2)

含义：
- 上轨：压力位
- 中轨：趋势线
- 下轨：支撑位
```

### 5.2 布林带形态

**布林收口**：
```
价格
 │    ╱╲
 │   ╱  ╲
 │  ╱    ╲    ← 价格波动收窄
 │ ╱      ╲
 │╱        ╲
 └─────────────
   上轨 ═══╗
         ═══╗
         ═══╝ 上轨收窄
   下轨 ═══╝

含义：波动收窄，变盘在即
操作：观望，等待突破方向
```

**布林开口**：
```
价格
 │        ╱
 │       ╱
 │      ╱   ← 价格突破
 │     ╱
 │    ╱
 └─────────────
   上轨 ═════╗
             ═══╗ 带宽放大
   下轨 ═════╝

含义：趋势启动
操作：顺势而为
```

### 5.3 布林带信号

**触及上轨**：
```
信号：
- 价格触及或突破上轨
- 量能放大
- 判断：超买或强势

两种情况：
1. 突破上轨 + 放量 + 强势 → 追涨
2. 触及上轨 + 缩量 + 回落 → 卖出
```

**触及下轨**：
```
信号：
- 价格触及或跌破下轨
- 量能萎缩
- 判断：超卖或弱势

两种情况：
1. 跌破下轨 + 放量 + 恐慌 → 观望
2. 触及下轨 + 缩量 + 支撑 → 买入
```

**中轨支撑/压力**：
```
中轨作用：
- 上升趋势：中轨是支撑位
- 下降趋势：中轨是压力位

操作：
- 价格回调至中轨 + 中轨向上 → 买入
- 价格反弹至中轨 + 中轨向下 → 卖出
```

## 六、指标组合使用

### 6.1 最佳组合信号

**强烈买入组合**：
```
条件：
1. MA5金叉MA10
2. MACD金叉
3. KDJ超卖区金叉

5. 布林下轨支撑

可靠性：⭐⭐⭐⭐⭐

```

**强烈卖出组合**：
```
条件：
1. MA5死叉MA10
2. MACD死叉
3. KDJ超买区死叉
4. RSI从超买区向下
5. 布林上轨压力

可靠性：⭐⭐⭐⭐⭐
准确率：>80%
```

### 6.2 指标矛盾处理

```
当指标信号不一致时：

1. 趋势优先
   - 上升趋势中，买入信号更可靠
   - 下降趋势中，卖出信号更可靠

2. 多数原则
   - 3个以上指标同向 → 跟随
   - 2:2分歧 → 观望

3. 量能确认
   - 放量配合的信号更可靠

4. 时间周期
   - 日线信号 > 60分钟信号 > 30分钟信号
```
