## 一、止损体系

### 1.1 为什么止损至关重要

```
数据统计：
- 亏损10% → 需上涨11%回本
- 亏损20% → 需上涨25%回本
- 亏损30% → 需上涨43%回本
- 亏损50% → 需上涨100%回本
- 亏损70% → 需上涨233%回本

结论：控制亏损比追求盈利更重要！

短线铁律：
截断亏损，让利润奔跑
```

### 1.2 固定止损

**百分比止损**：
```
单笔止损：≤ 3%
日内止损：≤ 5%
单周止损：≤ 10%
月度止损：≤ 15%

计算方法：
止损价 = 买入价 × (1 - 止损比例)

示例：
买入价：100元
止损比例：3%
止损价 = 100 × (1 - 0.03) = 97元
```

**金额止损**：
```
设定规则：
- 单笔最大亏损金额 = 总资金 × 1%
- 日内最大亏损金额 = 总资金 × 3%

示例：
总资金：100万
单笔最大亏损：1万
日内最大亏损：3万

执行：
- 单笔亏损达1万 → 止损
- 日内亏损达3万 → 停止交易
```

### 1.3 技术止损

**均线止损**：
```
MA5止损：
- 价格跌破MA5
- MA5走平或向下
- 短线止损信号

MA10止损：
- 价格跌破MA10
- 短线转弱
- 波段止损信号

操作：
跌破关键均线 → 考虑止损
收盘确认跌破 → 执行止损
```

**分时均线止损**：
```
信号：
1. 价格跌破分时均价线
2. 反弹无法站回
3. 黄线走平或向下

操作：
- 第一次跌破 → 警惕
- 反弹无力 → 减仓
- 确认跌破 → 止损
```

**支撑位止损**：
```
支撑位类型：
1. 前低点
2. 平台支撑
3. 布林下轨
4. 黄金分割位

操作：
- 价格跌破支撑位
- 反弹确认跌破
- 立即止损
```

**形态止损**：
```
M头止损：
- 价格跌破颈线
- 确认头部形态
- 立即止损

破位止损：
- 跌破整理平台
- 确认向下突破
- 止损离场

旗形破位：
- 跌破旗形下边
- 形态失败
- 止损
```

### 1.4 时间止损

**短线时间止损**：
```
规则：
- 买入后30分钟无盈利 → 关注
- 买入后1小时亏损 → 考虑止损
- 收盘前亏损 → 必须止损

逻辑：
- 如果判断正确，应该很快盈利
- 长时间不盈利说明判断失误
- 不与市场作对
```

**日内时间节点**：
```
关键时点：
10:00：确认当日方向
     - 亏损且方向不明 → 考虑止损

14:00：下午方向确认
     - 亏损且趋势不利 → 考虑止损

14:50：尾盘决策
     - 亏损 → 清仓
     - 不持亏过夜
```

### 1.5 止损执行要点

```
立即执行：
- 触发止损价 → 立即执行
- 不犹豫、不观望
- 不抱侥幸心理

分批止损：
- 大仓位 → 分批止损
- 第一批：触发价立即
- 第二批：确认跌破后

止损后：
- 不立即反向操作
- 复盘总结
- 等待下次机会
```

## 二、止盈体系

### 2.1 固定止盈

**百分比止盈**：
```
短线目标：
- 第一目标：+3%
- 第二目标：+5%
- 第三目标：+8%
- 极限目标：+10%

计算方法：
止盈价 = 买入价 × (1 + 止盈比例)

示例：
买入价：100元
第一目标 = 100 × 1.03 = 103元
第二目标 = 100 × 1.05 = 105元
第三目标 = 100 × 1.08 = 108元
```

**移动止盈**：
```
规则：
盈利3% → 止损移至成本价（保本）
盈利5% → 止损移至+2%
盈利8% → 止损移至+5%
盈利10% → 止损移至+8%

目的：
- 锁定部分利润
- 让利润奔跑
- 防止盈利回撤
```

### 2.2 技术止盈

**压力位止盈**：
```
压力位类型：
1. 前高点
2. 平台压力
3. 布林上轨
4. 黄金分割位

操作：
- 接近压力位 → 准备减仓
- 触及压力位 → 部分止盈
- 突破失败 → 全部止盈
```

**技术信号止盈**：
```
信号组合：
1. KDJ超买区死叉 → 强烈止盈
2. MACD顶背离 → 减仓或止盈
3. RSI>80转头 → 止盈
4. 布林触及上轨 → 止盈

判断：
- 信号越多 → 越应该止盈
- 信号越强 → 止盈比例越大
```

**量价异动止盈**：
```
放量滞涨：
- 成交量放大
- 价格不涨
- 主力出货
→ 止盈

冲高回落：
- 快速冲高
- 量能萎缩
- 上涨乏力
→ 止盈

尾盘异动：
- 尾盘拉升
- 做收盘价
- 次日低开
→ 尾盘止盈
```

### 2.3 分批止盈策略

**三段式止盈**：
```
计划：
第一目标（+3%）：卖出1/3
第二目标（+5%）：卖出1/2剩余
第三目标（+8%）：清仓

执行：
1. 到达第一目标
   - 卖出1/3仓位
   - 止损移至成本价

2. 到达第二目标
   - 卖出1/2剩余仓位
   - 止损移至+2%

3. 到达第三目标或信号出现
   - 清仓剩余仓位

优点：
- 锁定部分利润
- 防止踏空
- 降低心理压力
```

**动态止盈**：
```
规则：
盈利超过5%后，采用动态止盈

方法：
止盈价 = 最高价 - 回撤比例

示例：
买入价：100元
最高涨至110元
回撤比例：3%
动态止盈价 = 110 × 0.97 = 106.7元

执行：
- 价格从110回落至106.7 → 止盈
- 继续上涨至115 → 止盈价更新至111.55元

优点：
- 让利润奔跑
- 自动跟踪趋势
- 防止利润回撤过大
```

### 2.4 止盈心态管理

```
正确心态：
- 落袋为安
- 不贪婪
- 按计划执行
- 接受踏空

错误心态：
- 追求最高点
- 希望更多利润
- 不舍得卖
- 后悔卖早

调节方法：
- 设定目标，到价就卖
- 分批止盈，降低压力
- 踏空是常态，接受现实
```

## 三、仓位管理

### 3.1 单笔仓位控制

```
短线原则：
- 单笔仓位 ≤ 总资金10%
- 单笔最大亏损 ≤ 总资金1%
- 单笔最大盈利 ≥ 总资金2%

计算公式：
最大仓位 = 最大亏损金额 / (买入价 - 止损价)

示例：
总资金：100万
单笔最大亏损：1万
买入价：100元
止损价：97元（3%止损）

最大仓位 = 10000 / (100 - 97) = 3333股
投入金额 = 3333 × 100 = 33.3万（33%仓位）

但短线建议 ≤ 10%仓位，所以实际买入：
100万 × 10% = 10万 = 1000股
最大亏损 = 1000 × 3 = 3000元（0.3%）
```

### 3.2 日内仓位控制

```
总仓位控制：
- 日内总仓位 ≤ 总资金30%
- 单方向仓位 ≤ 总资金20%
- 保留现金 ≥ 总资金70%

分批建仓：
- 第一笔：计划仓位的50%
- 确认盈利后：再加30%
- 强势确认后：加满剩余20%

示例：
计划买入10万某股
- 第一笔：买入5万
- 盈利后：加仓3万
- 强势确认：加仓2万
```

### 3.3 持仓数量控制

```
短线持仓数量：
- 建议：3-5只
- 最少：1只（专注）
- 最多：不超过5只

原因：
- 太少：风险集中
- 太多：看不过来

分配原则：
- 主力仓位：60%（1-2只）
- 观察仓位：30%（1-2只）
- 现金：10%
```

## 四、风险预警系统

### 4.1 单笔风险预警

```
红色预警（立即止损）：
- 亏损达到止损线
- 技术形态破坏
- 利空消息

黄色预警（关注）：
- 亏损接近止损线
- 技术指标走弱
- 量价背离

绿色预警（持有）：
- 盈利中
- 技术指标健康
- 趋势向上
```

### 4.2 组合风险预警

```
红色预警（停止交易）：
- 日内亏损达5%
- 连续3笔亏损
- 组合整体亏损>10%

黄色预警（减仓观望）：
- 日内亏损达3%
- 连续2笔亏损
- 组合整体亏损>5%

绿色预警（正常交易）：
- 盈利或小亏
- 按计划执行
```

### 4.3 市场风险预警

```
系统性风险：
- 大盘跌幅>3%
- 板块集体下跌
- 利空消息
→ 减仓或清仓

流动性风险：
- 成交量骤降
- 买卖价差扩大
- 大单无法成交
→ 观望

情绪风险：
- 市场恐慌
- 非理性下跌
- 恐慌性抛售
→ 等待企稳
```

## 五、交易日记

### 5.1 记录模板

```
交易日记格式：

日期：2026-03-27
股票：中际旭创(300308)
操作：买入
价格：610元
数量：100股
仓位：10%
止损：590元(-3.3%)
止盈：640元(+5%)

【买入理由】
1. 技术面：MA5金叉MA10
2. 量能：量比1.8
3. 盘口：委比+45%

【风险点】
1. 大盘走弱
2. 前高630元压力

【执行情况】
- 10:05买入，价格610元
- 止损价设置590元
- 止盈目标640元

【结果】
- 14:30触及止盈640元
- 卖出100股
- 盈利：30元/股，合计3000元

【复盘】
- 操作正确
- 严格执行计划
- 无违规操作
```

```
每日复盘：
1. 检查交易执行情况
2. 分析盈亏原因
3. 总结经验教训
4. 优化交易系统

每周复盘：
1. 统计胜率和盈亏比
2. 分析连续亏损原因
3. 调整策略参数
4. 制定下周计划

每月复盘：
1. 月度收益统计
2. 最大回撤分析
3. 策略有效性评估
4. 风险控制改进
```
