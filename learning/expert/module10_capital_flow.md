### 2. 北向资金分析

**北向资金特征**：
- 价值投资，长期持有
- 偏好行业龙头
- 关注分红和ESG

**北向资金偏好行业**：
- ⭐⭐⭐⭐⭐ 消费（白酒、食品）
- ⭐⭐⭐⭐⭐ 医药
- ⭐⭐⭐⭐ 金融（银行、保险）
- ⭐⭐⭐⭐ 科技龙头

**北向资金信号**：

| 信号 | 含义 | 操作 |
|------|------|------|
| 连续5日净流入 | 强烈看多 | ✅ 积极布局 |
| 单日大额流入(>100亿) | 短期反弹 | ✅ 短期机会 |
| 连续5日净流出 | 看空 | 🔴 减仓 |
| 单日大额流出(>100亿) | 风险释放 | 🔴 回避 |

### 3. 主力资金流向

**主力资金分类**：

| 类型 | 金额 | 特点 |
|------|------|------|
| 超大单 | >100万 | 机构资金 |
| 大单 | 20-100万 | 大户资金 |
| 中单 | 5-20万 | 散户大户 |
| 小单 | <5万 | 散户资金 |

**主力资金四象限**：

| 主力资金 | 股价 | 判断 | 操作 |
|----------|------|------|------|
| 流入 | 上涨 | 建仓+拉升 | ✅ 买入 |
| 流入 | 下跌 | 洗盘吸筹 | ✅ 逢低布局 |
| 流出 | 上涨 | 拉高出货 | 🔴 减仓 |
| 流出 | 下跌 | 出货+下跌 | 🔴 卖出 |

**主力建仓特征**：
- 连续多日主力净流入
- 股价温和上涨或横盘
- 成交量逐步放大

**主力出货特征**：
- 连续多日主力净流出
- 股价快速拉升
- 成交量巨量，换手率>20%

### 4. 融资融券分析

**融资融券信号**：

| 指标 | 变化 | 含义 |
|------|------|------|
| 融资余额↑ | 上升 | 看多情绪增强 |
| 融资余额↓ | 下降 | 看多情绪减弱 |
| 融券余额↑ | 上升 | 看空情绪增强 |
| 融券余额↓ | 下降 | 看空情绪减弱 |

**融资融券比**：
```
融资买入额 / 融券卖出额

> 10：多头占优
< 5：空头占优
```

**综合判断**：
```
融资↑ + 融券↓ = 强烈看多 ✅
融资↓ + 融券↑ = 强烈看空 🔴
```

## 二、资金分析框架

### 综合分析流程

```
Step 1: 龙虎榜分析
  ├─ 机构买卖情况
  ├─ 游资动向
  └─ 北向资金参与

Step 2: 主力资金分析
  ├─ 近5日/20日流向
  ├─ 资金与股价关系
  └─ 建仓/出货判断

Step 3: 融资融券分析
  ├─ 融资余额变化
  ├─ 融券余额变化
  └─ 融资融券比

Step 4: 综合判断
  └─ 多信号共振 = 强信号
```

### 信号强度判断

| 信号组合 | 强度 | 操作 |
|----------|------|------|
| 机构买入 + 主力流入 + 融资增加 | ⭐⭐⭐⭐⭐ | 强烈买入 |
| 北向流入 + 主力流入 | ⭐⭐⭐⭐ | 买入 |
| 主力流入 + 融资增加 | ⭐⭐⭐⭐ | 买入 |
| 主力流出 + 融资减少 | 🔴🔴 | 卖出 |
| 机构卖出 + 主力流出 | 🔴🔴🔴 | 强烈卖出 |
