### 1. 四大经典因子

#### 价值因子

**指标**：

| 指标 | 判断标准 |
|------|----------|
| PE | <15低估 |
| PB | <1破净 |
| 股息率 | >4%高股息 |

**策略**：买入低估股票，长期持有

---

**指标**：

| 指标 | 判断标准 |

**策略**：买入高质量公司，稳健持有

#### 动量因子

| 指标 | 判断标准 |
|------|----------|
| 12月动量 | 前30%为强 |

**策略**：买入强势股，短线交易

### 2. 多因子模型

**综合评分**：
```
得分 = 价值×25% + 成长×25% + 质量×30% + 动量×20%
```

**因子筛选流程**：
```
1. 设定因子权重
2. 计算各因子得分
3. 加权得到综合得分
4. 选择综合得分最高的股票
5. 构建组合
```

**策略逻辑**：
```
1. PE < 30（低估）
2. 净利润增速 > 30%（成长）
3. ROE > 10%（质量）
4. 综合得分排序
5. 买入前10只
6. 每季度调仓
```

**综合评分**：
```
得分 = (1-PE分位)×30% + 增速分位×40% + ROE分位×30%
```
