## 一、核心知识掌握

### 1. 组合构建原则

**分散化原理**：
```
组合风险 < 个股风险加权平均
相关性越低，分散效果越好
15-20只股票可分散80%非系统性风险
```

**相关性矩阵**：

| 相关性 | 分散效果 | 示例 |
|--------|----------|------|
| **<0.3** | **极好** | 消费+科技 |
| 0.3-0.6 | 良好 | 新能源+医药 |
| 0.6-0.8 | 一般 | 同行业股票 |
| **>0.8** | **差** | 同行业龙头 |

**组合配置模板**：

| 类型 | 核心仓位 | 成长仓位 | 周期仓位 | 现金 |
|------|----------|----------|----------|------|
| 保守型 | 40% | 20% | 10% | 30% |
| **平衡型** | **50%** | **30%** | **10%** | **10%** |
| 进取型 | 20% | 60% | 15% | 5% |

### 2. 仓位管理策略

**凯利公式**：
```
最优仓位 = (胜率 × 盈亏比 - 败率) / 盈亏比

实际仓位 = 凯利仓位 × 安全系数（0.3）
```

**示例**：
```
胜率60%，盈亏比2
凯利仓位 = (0.6×2-0.4)/2 = 40%
实际仓位 = 40%×0.3 = 12%
```

**风险平价策略**：
```
权重_i = 1/波动率_i / Σ(1/波动率_j)

高波动股：低权重
低波动股：高权重
```

**仓位限制**：

| 类型 | 上限 |
|------|------|
| **单只股票** | **10-15%** |
| 单行业 | 30% |
| 核心仓位 | 50-60% |
| 成长仓位 | 20-30% |

### 3. 动态再平衡

**方法对比**：

| 方法 | 触发条件 | 优点 | 缺点 |
|------|----------|------|------|
| 定期再平衡 | 每季度 | 简单 | 可能错过时机 |
| 阈值再平衡 | 偏离>15% | 及时 | 需监控 |
| 战术配置 | 市场环境 | 灵活 | 需判断 |

**战术资产配置规则**：

| 市场情绪 | 调整方向 |
|----------|----------|
| 极度恐惧（<30） | 加仓+10-20% |
| 恐惧（30-45） | 加仓+5% |
| 中性（45-55） | 维持 |
| 贪婪（55-75） | 减仓-5% |
| 极度贪婪（>75） | 减仓-10-20% |

### 4. 组合绩效评估

**夏普比率**：
```
夏普比率 = (组合收益 - 无风险收益) / 组合波动率

>2：优秀 ✅
1-2：良好
0.5-1：一般
<0.5：差
```

**最大回撤**：
```
最大回撤 = (峰值 - 谷值) / 峰值

<10%：低风险 ✅
10-20%：中等风险
20-30%：较高风险
>30%：高风险 🔴
```

**信息比率**：
```
信息比率 = (组合收益 - 基准收益) / 跟踪误差

>0.5：优秀 ✅
0.3-0.5：良好
<0.3：一般
```
