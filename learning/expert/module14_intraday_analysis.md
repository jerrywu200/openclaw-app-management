## 一、分时图基础知识

### 1.1 分时图构成

```
分时图要素：
┌─────────────────────────────────────────────┐
│  股价曲线（白线）                            │
│      ╱╲                                     │
│     ╱  ╲   ╱╲                               │
│    ╱    ╲ ╱  ╲                              │
│───╱──────╲╱────均价线（黄线）───────────────│
│  ╱        ╲                                 │
│ ╱          ╲                                │
│                                             │
│  ▓▓▓▓░░▓▓▓▓░░░▓▓▓▓  成交量柱              │
└─────────────────────────────────────────────┘

核心要素：
├─ 白线：实时成交价格连线
├─ 黄线：分时均价线 = Σ(成交价×成交量)/Σ成交量
├─ 红柱：上涨成交量
├─ 绿柱：下跌成交量
└─ 涨跌幅：相对昨收的涨跌百分比
```

### 1.2 黄白线关系解读

| 情况 | 含义 | 操作建议 |
|------|------|----------|
| 白线在黄线上方 | 当前价格高于均价，多方占优 | 持有/买入 |
| 白线在黄线下方 | 当前价格低于均价，空方占优 | 谨慎/观望 |
| 黄白线粘合 | 价格在均价附近震荡 | 等待方向选择 |
| 白线远离黄线 | 短期偏离过大 | 注意回调风险 |

**判断口诀**：
```
白上黄下，多头发话
白下黄上，空头嚣张
黄白粘合，方向迷茫
远离黄线，回调在望
```

### 1.3 分时走势形态

#### 阶梯式上涨（健康）
```
价格
 │    ═══╗
 │       ═══╗
 │          ═══╗    ← 逐步抬高的平台
 │             ═══╗
 └───────────────── 时间

特征：
- 每次上涨后横盘整理
- 平台逐步抬高
- 量能温和
- 多头力量持续

操作：持有，等待下一波拉升
```

#### 直线拉升（需验证）
```
价格
 │                
 │              ╱
 │            ╱
 │          ╱
 │        ╱
 │      ╱
 └───────────────── 时间

两种可能：
1. 真拉升：量能持续放大，不回落 → 追涨
2. 诱多：量能萎缩，快速回落 → 不追

判断标准：
- 涨幅>3%且量比>2 → 真拉升
- 涨幅<2%且量能萎缩 → 诱多
```

#### 震荡下行（弱势）
```
价格
 │╗
 │ ╲
 │  ╲╗
 │   ╲
 │    ╲╗
 │     ╲
 └───────────────── 时间

特征：
- 反弹无力
- 低点逐步降低
- 均线压制
- 空头占优

操作：观望或止损
```

#### V型反转（情绪转折）
```
价格
 │╗            ╱╗
 │ ╲          ╱  
 │  ╲        ╱    
 │   ╲      ╱      
 │    ╲    ╱       
 │     ╲  ╱        
 └───────────────── 时间

特征：
- 快速下跌后快速拉升
- 量能明显放大
- 情绪转折点

操作：确认突破后介入
```

## 二、盘口分析

### 2.1 五档盘口

```
┌─────────────────────────────────┐
│ 卖5  │ 100.50 │  500手  │      │
│ 卖4  │ 100.40 │  300手  │      │
│ 卖3  │ 100.30 │  200手  │ 卖盘 │
│ 卖2  │ 100.20 │  150手  │      │
│ 卖1  │ 100.10 │  100手  │ ←最近的卖单
├─────────────────────────────────┤
│ 买1  │ 100.00 │  200手  │ ←最近的买单
│ 买2  │  99.90 │  300手  │      │
│ 买3  │  99.80 │  250手  │ 买盘 │
│ 买4  │  99.70 │  180手  │      │
│ 买5  │  99.60 │  400手  │      │
└─────────────────────────────────┘

解读：
- 买1-买5：当前挂买单（想买的）
- 卖1-卖5：当前挂卖单（想卖的）
- 价格越接近，越容易成交
```

### 2.2 委比与委差

**委比公式**：
```
委比 = (买盘总量 - 卖盘总量) / (买盘总量 + 卖盘总量) × 100%
```

**判断标准**：

| 委比范围 | 强弱 | 含义 |
|----------|------|------|
| >50% | 很强 | 买盘远大于卖盘，看涨 |
| 20%~50% | 偏强 | 买盘占优 |
| -20%~20% | 平衡 | 多空均衡 |
| -50%~-20% | 偏弱 | 卖盘占优 |
| <-50% | 很弱 | 卖盘远大于买盘，看跌 |

**委差**：
```
委差 = 买盘总量 - 卖盘总量
```
- 委差为正：买盘强
- 委差为负：卖盘强

### 2.3 内外盘

**定义**：
- **外盘**：主动买入成交（买方主动以卖价成交）→ 看多
- **内盘**：主动卖出成交（卖方主动以买价成交）→ 看空

**判断标准**：

| 情况 | 含义 | 操作建议 |
|------|------|----------|
| 外盘 >> 内盘 | 主动买盘多，多头强 | 看涨 |
| 内盘 >> 外盘 | 主动卖盘多，空头强 | 看跌 |
| 外盘 ≈ 内盘 | 多空平衡 | 观望 |

**口诀**：
```
外盘大，买气旺，股价上涨有力量
内盘大，卖压强，股价下跌要提防
外内平，多空忙，震荡整理看方向
```

### 2.4 大单异动

**大买单（吸筹/拉升信号）**：
```
特征：
- 连续出现大买单（>500手）
- 买1-买5量突然增加
- 卖盘被快速吃掉

判断：
- 股价低位出现 → 主力吸筹
- 股价高位出现 → 可能诱多
```

**大卖单（出货信号）**：
```
特征：
- 连续出现大卖单
- 卖1-卖5量突然增加
- 买盘被快速打掉

判断：
- 股价高位出现 → 主力出货
- 股价低位出现 → 可能诱空
```

**托单（诱多）**：
```
特征：
- 买2-买5有大量挂单
- 但股价不涨
- 小单不断卖出

含义：主力在下方托单，吸引散户买入，实则出货
```

**压单（诱空）**：
```
特征：
- 卖2-卖5有大量挂单
- 但股价不跌
- 小单不断买入

含义：主力在上方压单，恐吓散户卖出，实则吸筹
```

## 三、日内节奏分析

### 3.1 时间分段（A股）

| 时间段 | 名称 | 特点 | 操作策略 |
|--------|------|------|----------|
| 9:15-9:25 | 集合竞价 | 决定开盘价 | 观察量价判断强弱 |
| 9:30-10:00 | 开盘半小时 | 最活跃，方向确认 | 确认当日趋势 |
| 10:00-11:30 | 上午盘 | 趋势延续或震荡 | 顺势操作 |
| 11:30-13:00 | 午休 | 市场休市 | 复盘，制定下午策略 |
| 13:00-14:00 | 下午开盘 | 常有方向选择 | 关注突破信号 |
| 14:00-14:30 | 下午盘 | 趋势延续 | 谨慎追涨杀跌 |
| 14:30-15:00 | 尾盘 | 异动频发 | 关注尾盘信号 |

### 3.2 关键时点操作

#### 集合竞价（9:15-9:25）
```
观察要点：
1. 竞价量能
   - 量比>2：关注度高
   - 量比<1：关注度低

2. 开盘价位置
   - 高开>3%：强势，可能继续涨
   - 高开1-3%：正常强势
   - 平开：观望
   - 低开1-3%：正常弱势
   - 低开>3%：弱势，可能恐慌

3. 竞价形态
   - 竞价逐步抬高：多头积极
   - 竞价逐步走低：空头压制
```

#### 开盘半小时（9:30-10:00）
```
重要性：决定当日基调

三种走势：
1. 高开高走
   - 开盘后继续上涨
   - 量能放大
   - 判断：强势，可跟进

2. 高开低走
   - 开盘后快速回落
   - 跌破开盘价
   - 判断：诱多或冲高回落，不追

3. 低开高走
   - 开盘后快速拉升
   - 突破开盘价
   - 判断：可能有行情，关注

操作原则：
- 9:35之前不急于操作
- 10:00确认方向后再决策
```

#### 尾盘半小时（14:30-15:00）
```
尾盘异动类型：

1. 尾盘拉升
   - 14:30后快速上涨
   - 量能放大
   - 含义：次日看涨
   - 操作：可持有过夜

2. 尾盘跳水
   - 14:30后快速下跌
   - 量能放大
   - 含义：次日看跌
   - 操作：减仓或清仓

3. 尾盘横盘
   - 价格波动小
   - 含义：多空平衡
   - 操作：观望

注意：
- 14:50后拉升要警惕（可能是做盘）
- 14:50后下跌要重视（可能是真实卖压）
```

## 四、分时买卖点

### 4.1 买入信号

```
1. 黄线支撑买入
   - 价格回调至黄线附近
   - 黄线向上
   - 量能萎缩后放大
   → 买入

2. 突破前高买入
   - 价格突破分时前高
   - 量能配合
   - 黄线支撑
   → 买入

3. V型反转买入
   - 快速下跌后企稳
   - 开始回升
   - 量能放大
   → 确认突破后买入

4. 开盘强势买入
   - 高开 + 量比>2
   - 5分钟内站稳均价线
   - 委比>50%
   → 可追涨
```

### 4.2 卖出信号

```
1. 跌破黄线卖出
   - 价格跌破均价线
   - 黄线走平或向下
   - 量能放大
   → 卖出

2. 冲高回落卖出
   - 冲高后无法维持
   - 跌破分时均线
   - 量能萎缩
   → 卖出

3. 尾盘跳水卖出
   - 14:30后快速下跌
   - 量能放大
   - 跌破关键支撑
   → 卖出

4. 技术位卖出
   - 触及日内目标价
   - 量能异常放大
   - 涨幅过大（>5%）
   → 分批卖出
```
