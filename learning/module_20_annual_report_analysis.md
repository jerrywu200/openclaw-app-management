## Day 1: 财务数据变化分析

### 核心知识点

#### 1.1 营收/利润增速判断

```
判断标准：
├─ 营收增速 > 30%：高速增长
├─ 营收增速 15-30%：快速增长
├─ 营收增速 5-15%：稳定增长
├─ 营收增速 0-5%：缓慢增长
└─ 营收增速 < 0：负增长

利润增速 vs 营收增速：
├─ 利润增速 > 营收增速 → 经营效率提升 ✅
├─ 利润增速 ≈ 营收增速 → 正常
└─ 利润增速 < 营收增速 → 成本压力增大 🔴
```

**关键公式**：
```
营收增长率 = (本期营收 - 上期营收) / 上期营收 × 100%
净利润增长率 = (本期净利润 - 上期净利润) / 上期净利润 × 100%
```

#### 1.2 分季度业绩拆解

```
分析维度：
├─ Q1-Q4营收分布 → 识别季节性规律
├─ 季度环比增速 → 判断趋势变化
├─ 季度同比增速 → 剔除季节因素
└─ 单季净利润 → 发现业绩拐点

判断标准：
├─ Q4通常是旺季或淡季？
├─ 季度波动是否正常？
└─ 是否有单季异常？
```

#### 1.3 毛利率/净利率趋势

```
毛利率变化原因：
├─ 价格变化（提价/降价）
├─ 成本变化（原材料/人工）
├─ 产品结构变化
└─ 行业竞争格局变化

净利率变化原因：
├─ 毛利率变化
├─ 费用率变化（销售/管理/研发/财务）
├─ 税率变化
└─ 非经常性损益
```

#### 1.4 关键财务指标追踪

| 指标类型 | 核心指标 | 健康标准 |
|----------|----------|----------|
| **盈利能力** | ROE | >15%良好，>20%优秀 |
| **盈利能力** | ROA | >5%良好，>10%优秀 |
| **偿债能力** | 资产负债率 | <60%健康 |
| **偿债能力** | 流动比率 | >1.5健康 |
| **运营效率** | 存货周转天数 | 行业对比 |
| **运营效率** | 应收周转天数 | <60天良好 |

### 实战案例

**比亚迪2025年报**：
- 营收增速：+3.46%（大幅放缓）
- 净利润增速：-19%（首次下滑）
- 判断：🔴 利润增速远低于营收增速，盈利能力恶化

**洛阳钼业2025年报**：
- 营收增速：-2.98%
- 净利润增速：+50.30%
- 判断：✅ 利润增速远超营收增速，盈利能力提升

### 学习收获

- 掌握了增速判断标准
- 学会了季度拆解方法
- 理解了毛利率变化的含义
- 能够追踪关键财务指标变化

## Day 2: 业务定性分析（MD&A解读）

### 核心知识点

#### 2.1 核心战略解读

```
分析维度：
├─ 战略方向是否清晰
├─ 战略是否可执行
├─ 战略是否有差异化
└─ 战略与财务是否匹配

判断标准：
✅ 好的战略：清晰 + 可执行 + 有差异化 + 有护城河
🔴 差的战略：模糊 + 口号化 + 随波逐流
```

#### 2.2 市占率变化

```
分析维度：
├─ 市占率绝对值（行业龙头？前三？）
├─ 市占率变化趋势（上升/下降/稳定）
├─ 市占率变化原因
└─ 市占率与定价权关系

关键问题：
├─ 市占率是否达到垄断临界点（30%+）？
├─ 市占率提升是靠价格战还是产品力？
└─ 市占率变化是否可持续？
```

#### 2.3 新产品/新产能进度

```
分析维度：
├─ 新产品类型和预期贡献
├─ 新产能规模和资金来源
├─ 产能利用率
└─ 投资回报预期

判断标准：
✅ 好的产能扩张：产能利用率高 + 市场需求旺盛 + 资金充裕
🔴 差的产能扩张：产能利用率低 + 行业产能过剩 + 高负债扩张
```

#### 2.4 管理层质量评估

| 维度 | 评估标准 | 观察指标 |
|------|----------|----------|
| 战略眼光 | 是否提前布局行业趋势 | 过往战略执行情况 |
| 执行能力 | 目标达成率 | 历史承诺兑现情况 |
| 沟通透明度 | 年报披露是否详细 | 投资者沟通频率 |
| 股东回报 | 分红政策是否稳定 | 分红率变化趋势 |

#### 2.5 行业竞争格局变化

```
分析维度：
├─ 行业集中度变化（CR3/CR5）
├─ 龙头份额变化
├─ 新进入者威胁
├─ 价格战情况
└─ 替代品威胁

判断标准：
✅ 行业格局改善：龙头份额提升 + 集中度提高 + 价格稳定
🔴 行业格局恶化：价格战 + 新进入者 + 龙头份额下降
```

### 实战案例

**比亚迪2025年报**：
- 战略：全球化、高端化、技术领先 ✅ 清晰可执行
- 市占率：全球新能源冠军、全球销量第五 ✅ 持续提升
- 新产能：海外工厂加速布局 ✅ 产能释放
- 管理层：战略执行到位，但盈利需改善 🟡

**洛阳钼业2025年报**：
- 战略：铜金两极、产能扩张 ✅ 明确清晰
- 市占率：全球第一大钴生产商 ✅ 定价权强
- 新产能：KFM二期2027年投产 ✅ 成长路径清晰
- 管理层：超额完成产量指引 ✅ 执行力强

### 学习收获

- 理解了MD&A的重要性（数字是结果，业务是原因）
- 掌握了战略解读框架
- 学会了市占率变化分析
- 能够评估管理层质量

## Day 3: 现金流与资本配置

### 核心知识点

#### 3.1 自由现金流（FCF）计算与判断

```
FCF计算公式：
FCF = 经营现金流 - 资本开支(Capex)

判断标准：
├─ FCF > 0 且增长 → 现金创造能力强 ✅
├─ FCF > 0 但下降 → 现金创造能力减弱 🟡
├─ FCF < 0 但经营现金流 > 0 → 高投入期 🟡
└─ FCF < 0 且经营现金流 < 0 → 现金危机 🔴

利润质量判断：
FCF/净利润 > 0.8 → 利润质量高 ✅
FCF/净利润 0.5-0.8 → 利润质量尚可
FCF/净利润 < 0.5 → 利润质量差 🔴
```

#### 3.2 资本开支(Capex)趋势

```
分析维度：
├─ Capex金额变化
├─ Capex/营收比率
├─ Capex资金来源
└─ Capex投资方向

判断标准：
├─ Capex增速 > 营收增速 → 高投入扩张期
├─ Capex增速 < 营收增速 → 投资收缩期
└─ Capex/营收 > 20% → 高投入期

关键问题：
├─ Capex是否有足够现金流支持？
├─ Capex投资回报率(ROIC)是否>WACC？
└─ Capex方向是否匹配战略？
```

#### 3.3 分红政策评估

```
分析维度：
├─ 分红率变化趋势
├─ 分红稳定性（连续性）
├─ 分红与成长性平衡
└─ 分红资金来源

健康分红标准：
├─ 分红率30-50%
├─ 连续多年分红
├─ 有足够留存支持发展
└─ 分红金额稳定或增长

A股常见分红率：
├─ 高成长科技股：10-20%
├─ 稳定消费股：30-50%
├─ 公用事业股：50-70%
└─ 银行股：30%左右
```

#### 3.4 股东回报分析

```
资本配置去向分析：
├─ 资本开支（产能扩张）
├─ 研发投入（技术创新）
├─ 分红（股东回报）
├─ 回购（提升每股收益）
└─ 储备现金（安全边际）

资本配置效率评估：
ROIC = 税后经营利润 / 投入资本
├─ ROIC > WACC → 创造价值 ✅
└─ ROIC < WACC → 毁灭价值 🔴
```

### 实战案例

**比亚迪2025年报**：
- FCF：-977亿 🔴 高投入期，资本开支远超经营现金流
- Capex：1568亿，主要用于产能扩张
- 分红：218亿，股利支付率28.7% ✅
- 判断：高投入期，需关注投入产出比

**洛阳钼业2025年报**：
- FCF：+39亿 ✅ 自由现金流为正
- Capex：169亿，适中水平
- 分红：55亿，股利支付率40% ✅
- 判断：财务稳健，可持续扩张

### 学习收获

- 掌握了FCF计算和判断方法
- 理解了资本配置的重要性
- 学会了分红政策评估
- 能够评估股东回报质量

## Day 4: 前瞻性指引分析

### 核心知识点

#### 4.1 管理层业绩指引

```
指引类型：
├─ 定量指引：营收增长目标、利润增长目标
└─ 定性指引：行业趋势判断、战略重点方向

分析维度：
├─ 指引的明确性
├─ 指引的可信度（历史达成率）
├─ 指引的保守/激进程度
└─ 指引与行业趋势匹配度
```

#### 4.2 指引vs市场预期

```
对比方法：
├─ 获取研报EPS预测作为市场预期
├─ 对比实际业绩与市场预期
└─ 分析偏差原因

判断标准：
├─ 实际 > 预期10%+ → 超预期 ✅ 正面信号
├─ 实际 ≈ 预期 → 符合预期
└─ 实际 < 预期10%+ → 低于预期 🔴 负面信号

股价反应预测：
├─ 指引超预期 → 股价可能上涨
├─ 指引符合预期 → 股价平稳
└─ 指引低于预期 → 股价可能下跌
```

#### 4.3 战略规划解读

```
分析维度：
├─ 3-5年战略规划清晰度
├─ 战略里程碑是否明确
├─ 战略资源是否匹配
└─ 战略差异化程度

判断标准：
✅ 好的战略规划：清晰 + 有里程碑 + 有资源匹配
🔴 差的战略规划：模糊 + 只有口号 + 无资源支持
```

#### 4.4 行业趋势预判

```
对比分析：
├─ 管理层行业判断
├─ 外部研究报告判断
└─ 历史判断准确性

一致性分析：
├─ 管理层与外部研究一致 → 可信度高 ✅
├─ 管理层与外部研究分歧 → 需深入分析 🟡
└─ 管理层过往判断偏差大 → 需谨慎 🔴
```

### 实战案例

**比亚迪2025年报**：
- 2026海外目标：130万辆（+24%）✅ 明确
- 整体增速指引：不明确 🟡
- 研报预测EPS：5.07元（隐含+41%增长）
- 实际EPS：3.58元 🔴 低于预期

**洛阳钼业2025年报**：
- 2026铜目标：80万吨（+8%）✅ 明确
- KFM二期：2027年投产 ✅ 里程碑清晰
- 研报预测EPS：0.91元
- 实际EPS：0.95元 ✅ 超预期

### 学习收获

- 理解了前瞻指引的重要性
- 掌握了指引vs市场预期对比方法
- 学会了战略规划解读
- 能够判断管理层判断的可信度

## Day 5: 红旗信号检测与估值逻辑链条

### 核心知识点

#### 5.1 红旗信号检测

| 信号类型 | 检测方法 | 风险等级 |
|----------|----------|----------|
| 利润质量 | 净现比 = 经营现金流/净利润 | <0.5为差 |
| 应收异常 | 应收增速 vs 营收增速 | 应收>营收1.5倍为异常 |
| 存货积压 | 存货周转天数变化 | 增加>30天需关注 |
| 商誉风险 | 商誉/总资产 | >20%风险高 |
| 关联交易 | 年报披露审查 | 异常关联交易 |

#### 5.2 估值逻辑链条

```
目标价调整 = EPS变化 × 估值倍数变化

EPS变化原因分解：
├─ 营收增长预测调整
├─ 毛利率预测调整
├─ 费用率预测调整
└─ 其他因素

估值倍数变化原因：
├─ 增长预期变化
├─ 行业估值变化
├─ 公司风险变化
└─ 市场情绪变化
```

#### 5.3 估值调整分解表

| 因素 | 原预测 | 新预测 | 变化 | 贡献度 |
|------|--------|--------|------|--------|
| EPS | XX元 | XX元 | XX% | XX% |
| PE倍数 | XX倍 | XX倍 | XX% | XX% |
| 目标价 | XX元 | XX元 | XX% | 100% |

### 实战案例

**比亚迪估值调整**：
- 原EPS预测：5.07元
- 新EPS：3.58元（-29%）
- 原PE：22倍 → 新PE：20倍（-9%）
- 目标价：111元 → 90元（-19%）

**洛阳钼业估值调整**：
- 原EPS预测：0.91元
- 新EPS：0.95元（+4%）
- PE维持18倍
- 目标价：20-22元（当前低估）

### 学习收获

- 掌握了红旗信号检测方法
- 理解了估值逻辑链条
- 学会了目标价调整分解
