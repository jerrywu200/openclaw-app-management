# 实战复盘积累 - 学习模块

> 更新日期: 2026-03-27
> 目标: 通过复盘验证分析方法，沉淀实战经验

---

## 学习目标

1. **验证分析框架** - 检验评分系统的有效性
2. **总结成功经验** - 识别正确判断的关键因素
3. **提炼失败教训** - 发现分析盲点和偏差
4. **完善方法论** - 持续优化分析框架

---

## 复盘方法

### 一、复盘流程

```
┌─────────────────────────────────────────────────────────────┐
│  1. 选取复盘标的                                             │
│     └─ 从已分析股票中选择代表性案例                          │
│                                                             │
│  2. 回顾原始分析                                             │
│     └─ 评级、评分、关键结论、建议操作                        │
│                                                             │
│  3. 对比实际走势                                             │
│     └─ 价格变化、量价配合、关键事件                          │
│                                                             │
│  4. 检验关键判断                                             │
│     └─ 正确判断 → 总结成功因素                               │
│     └─ 错误判断 → 分析偏差原因                               │
│                                                             │
│  5. 提炼经验教训                                             │
│     └─ 更新方法论、补充案例库                                │
└─────────────────────────────────────────────────────────────┘
```

### 二、复盘周期

| 周期 | 内容 | 频率 |
|------|------|------|
| 短期复盘 | 3-5个交易日走势验证 | 每周 |
| 中期复盘 | 1-3个月走势验证 | 每月 |
| 年度复盘 | 全年投资总结 | 年底 |
