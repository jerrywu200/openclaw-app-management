## 行业一：新能源汽车行业

### 一、行业概况

```
市场规模：
├─ 全球：约 1500 万辆/年
├─ 中国：约 900 万辆/年（占全球60%）
├─ 增速：约 30-40%
└─ 渗透率：约 35%（中国）

发展阶段：成长期
├─ 渗透率 35% → 成长中期
├─ 增速 30%+ → 高增长
└─ 天花板：替代燃油车（渗透率可达 70%+）
```

### 二、产业链分析

```
┌─────────────────────────────────────────────────┐
│  上游：原材料                                    │
│  ├─ 锂矿：天齐锂业、赣锋锂业                     │
│  ├─ 钴镍：华友钴业                               │
│  └─ 稀土：北方稀土                               │
├─────────────────────────────────────────────────┤
│  中游：零部件                                    │
│  ├─ 动力电池：宁德时代、比亚迪、国轩高科         │
│  ├─ 电机电控：汇川技术、精进电动                 │
│  ├─ 热管理：三花智控                             │
│  └─ 车身压铸：旭升集团、文灿股份                 │
├─────────────────────────────────────────────────┤
│  下游：整车                                      │
│  ├─ 自主品牌：比亚迪、吉利、长城                 │
│  ├─ 新势力：理想、蔚来、小鹏                     │
│  └─ 传统转型：上汽、广汽                         │
└─────────────────────────────────────────────────┘

价值分布：
├─ 动力电池：占整车成本 30-40%
├─ 电机电控：占整车成本 10-15%
└─ 利润主要在电池和整车
```

### 三、竞争格局（波特五力）

```
现有竞争者：激烈 ⚠️⚠️
├─ 玩家众多：比亚迪、特斯拉、新势力、传统车企
├─ 价格战：特斯拉降价引发行业价格战
├─ 差异化：技术、品牌、渠道
└─ 退出壁垒高

潜在进入者：中等
├─ 小米汽车已入局
├─ 华为合作模式
└─ 技术和资金门槛高

替代品威胁：弱 ✅
├─ 燃油车被替代
└─ 氢能、飞行汽车远期

供应商议价：强 ⚠️
├─ 电池：宁德时代议价能力强
├─ 锂矿：上游资源集中
└─ 芯片：部分依赖进口

购买者议价：中等
├─ 消费者选择多
├─ 品牌忠诚度一般
└─ 价格敏感
```

### 四、商业模式分析

```
比亚迪商业模式：

价值主张：
└─ 性价比高的新能源汽车

核心竞争力：
├─ 垂直整合（电池+芯片+整车）✅
├─ 规模效应（全球第一）✅
├─ 技术积累（刀片电池、DM混动）✅

├─ 整车销售
├─ 电池外供
```

### 五、投资机会与风险

```

├─ 行业高增长持续
├─ 龙头份额提升
├─ 海外市场拓展
└─ 产品高端化
```

## 行业五：金融行业

### 一、行业概况

```
银行业：
├─ 总资产：约 350 万亿元
├─ 净利润：约 2.5 万亿元
├─ 增速：约 5%
└─ ROE：约 10-12%

保险业：
├─ 保费收入：约 5 万亿元
├─ 增速：约 5-8%
└─ 渗透率：较低

证券业：
├─ 行业规模：约 5000 亿元
├─ 周期性强
└─ 与资本市场相关

发展阶段：成熟期
├─ 增速放缓
├─ 竞争格局稳定
└─ 监管严格
```

### 二、商业模式分析

```
银行：
├─ 盈利模式：存贷利差 + 中间业务
├─ ROE驱动：净息差 × 杠杆
├─ 核心指标：不良率、拨备覆盖率
└─ 护城河：牌照 + 渠道

保险：
├─ 盈利模式：保费 + 投资
├─ 寿险：长期价值
├─ 财险：承保盈利
└─ 护城河：渠道 + 品牌

券商：
├─ 盈利模式：佣金 + 投行 + 自营
├─ 周期性强
└─ 与市场行情相关
```

### 三、投资机会与风险

```
投资机会：
├─ 估值低（PB 0.5-0.8）✅
├─ 分红稳定
├─ 经济复苏受益
└─ 适合防御配置

风险因素：
├─ 经济下行风险 ⚠️
├─ 利率下行压缩息差
├─ 不良资产风险
└─ 监管趋严

├─ 银行：招商银行、宁波银行
```
