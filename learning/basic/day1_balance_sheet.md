## 一、资产负债表的本质

### 核心等式
```
资产 = 负债 + 所有者权益

资产：公司拥有的资源
负债：公司欠别人的钱
所有者权益：股东投入 + 累计盈利
```

### 形象理解
```
资产负债表就像一张"照片"
记录公司在某一时点的财务状况

资产 = 钱去哪了（占用）
负债+权益 = 钱从哪来（来源）
```

---

## 二、资产结构分析

### 流动资产（1年内变现）

| 项目 | 含义 | 分析要点 |
|------|------|----------|
| **货币资金** | 现金+银行存款 | 越多越好，但过多可能资金效率低 |
| **应收账款** | 客户欠的钱 | 占营收比例<20%，增长要与营收匹配 |
| **存货** | 待售商品 | 周转率要稳定，积压是风险信号 |
| **预付款项** | 提前付的钱 | 过高说明议价能力弱 |

### 非流动资产（长期持有）

| 项目 | 含义 | 分析要点 |
|------|------|----------|
| **固定资产** | 厂房、设备 | 重资产行业占比高，关注折旧 |
| **无形资产** | 专利、商标 | 科技公司重要，警惕虚高 |
| **商誉** | 并购溢价 | 过高有减值风险 |
| **长期投资** | 股权投资 | 关注投资质量和收益 |

---

## 三、负债结构分析

### 流动负债（1年内偿还）

| 项目 | 含义 | 分析要点 |
|------|------|----------|
| **短期借款** | 银行贷款 | 过高有流动性风险 |
| **应付账款** | 欠供应商的钱 | 适度高说明议价能力强 |
| **预收款项** | 客户预付 | 好事！说明产品抢手 |

### 非流动负债（长期偿还）

| 项目 | 含义 | 分析要点 |
|------|------|----------|
| **长期借款** | 长期贷款 | 关注用途和还款能力 |
| **应付债券** | 发行的债券 | 利率和到期日 |

---

## 四、关键分析指标

### 1. 偿债能力

```
流动比率 = 流动资产 / 流动负债
- > 2：很强
- 1.5-2：健康
- 1-1.5：需关注
- < 1：危险（流动资产不足以覆盖流动负债）

速动比率 = (流动资产 - 存货) / 流动负债
- > 1：健康
- 剔除存货，更严格地测试偿债能力
```

### 2. 资产负债率

```
资产负债率 = 总负债 / 总资产 × 100%

- < 40%：保守，财务稳健
- 40%-60%：适中，合理杠杆
- 60%-70%：偏高，需关注
- > 70%：风险较大
```

### 3. 应收账款分析

```
应收账款周转率 = 营业收入 / 平均应收账款
应收账款周转天数 = 365 / 周转率

周转天数越短越好：
- < 30天：优秀
- 30-60天：正常
- 60-90天：需关注
- > 90天：回收慢，风险大
```

### 4. 存货分析

```
存货周转率 = 营业成本 / 平均存货
存货周转天数 = 365 / 周转率

存货周转天数越短越好：
- 行业差异大，需对比同行
- 周转天数增加 = 存货积压风险
```

## 五、红旗信号识别

### 资产负债表异常信号

| 异常信号 | 可能问题 |
|----------|----------|
| 应收账款快速增长，超过营收增速 | 收入质量下降，可能虚增收入 |
| 存货持续增加，周转率下降 | 产品滞销，可能计提减值 |
| 商誉占比过高 | 并购溢价，减值风险大 |
| 预付款项异常增加 | 可能资金挪用 |
| 其他应收款过多 | 可能关联方占用资金 |
| 资产负债率持续上升 | 债务压力加大 |
| 短期借款远大于长期借款 | 短贷长投，流动性风险 |

## 六、实践案例：贵州茅台

### 资产负债表特点（简化分析）

```
贵州茅台资产负债表特点：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

资产端：
├─ 货币资金占比极高（>50%）✅ 现金充裕
├─ 存货占比适中（基酒储存需要）✅ 正常
├─ 固定资产占比适中 ✅ 轻资产
└─ 无商誉 ✅ 无并购风险

负债端：
├─ 负债率极低（<30%）✅ 财务稳健
├─ 预收款项/合同负债高 ✅ 产品抢手
├─ 几乎无有息负债 ✅ 无偿债压力
└─ 应付账款适中 ✅ 议价能力强

结论：财务极其健康，"现金奶牛"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 七、学习总结

### 资产负债表分析流程

```
1. 看结构
   ├── 资产构成（重资产vs轻资产）
   └── 负债构成（有息vs无息）

2. 看变化
   ├── 同比变化
   └── 占比变化趋势

3. 算指标
   ├── 流动比率/速动比率
   ├── 资产负债率
   ├── 应收账款周转率
   └── 存货周转率

4. 找异常
   ├── 红旗信号
   └── 与同行对比

5. 下结论
   ├── 财务健康度
   └── 主要风险点
```

### 核心要点记忆

```
✅ 资产负债表是"时点照片"
✅ 资产 = 负债 + 权益（恒等）
✅ 流动比率 > 1.5 较安全
✅ 资产负债率 < 60% 较健康
✅ 应收账款增速 ≤ 营收增速
✅ 存货周转率要稳定或上升
✅ 预收款项多是好事
✅ 商誉过高要警惕减值
```
