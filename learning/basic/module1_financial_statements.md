## Day 1：资产负债表

### 学习材料

#### 📚 必读教材
1. **《财务智慧》- Karen Berman**
   - 章节：资产负债表部分
   - 时长：约2小时
   - 重点：资产结构、负债分析

2. **《一本书读懂财报》- 肖星**
   - 章节：资产负债表
   - 时长：约1.5小时
   - 重点：通俗易懂的解释

#### 🎥 推荐视频
1. **清华公开课《财务分析与决策》- 肖星**
   - 第2-3讲：资产负债表
   - 时长：约1小时
   - 链接：网易公开课

#### 📝 核心概念

```
资产负债表核心结构
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

资产 = 负债 + 所有者权益

┌─────────────────────────────────────┐
│  资产                               │
├─────────────────────────────────────┤
│  流动资产：货币资金、应收账款、存货  │
│  非流动资产：固定资产、无形资产      │
└─────────────────────────────────────┘
                    = 
┌─────────────────────────────────────┐
│  负债                               │
├─────────────────────────────────────┤
│  流动负债：短期借款、应付账款        │
│  非流动负债：长期借款、债券          │
└─────────────────────────────────────┘
        + 
┌─────────────────────────────────────┐
│  所有者权益                         │
├─────────────────────────────────────┤
│  实收资本、资本公积、盈余公积        │
│  未分配利润                         │
└─────────────────────────────────────┘
```

#### 🔑 关键分析点

| 分析维度 | 关注指标 | 健康信号 | 警惕信号 |
|----------|----------|----------|----------|
| 偿债能力 | 流动比率 | >1.5 | <1 |
| 资产结构 | 固定资产占比 | 合理 | 过高/过低 |
| 应收账款 | 占营收比例 | <20% | 快速增长 |
| 存货 | 存货周转率 | 稳定 | 积压上升 |
| 负债水平 | 资产负债率 | <60% | >70% |

#### 📊 实践任务

1. **案例分析：贵州茅台资产负债表**
   - 下载近3年财报
   - 分析资产结构变化
   - 评估财务健康度
   - 输出500字分析笔记

2. **对比分析：茅台 vs 五粮液**
   - 对比资产结构差异
   - 分析负债水平差异
   - 判断哪家财务更稳健

## Day 2：利润表

### 学习材料

#### 📚 必读教材
1. **《财务智慧》- Karen Berman**
   - 章节：利润表部分
   - 时长：约2小时

2. **《一本书读懂财报》- 肖星**
   - 章节：利润表
   - 时长：约1.5小时

3. **《手把手教你读财报》- 唐朝**
   - 章节：利润表分析
   - 时长：约2小时
   - 重点：实操性强

#### 🎥 推荐视频
1. **清华公开课《财务分析与决策》- 肖星**
   - 第4-5讲：利润表
   - 时长：约1小时

#### 📝 核心概念

```
利润表结构
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

营业收入
  - 营业成本
─────────────────
= 毛利润 → 毛利率 = 毛利/营收

  - 销售费用
  - 管理费用
  - 研发费用
  - 财务费用
─────────────────
= 营业利润

  + 营业外收入
  - 营业外支出
─────────────────
= 利润总额

  - 所得税
─────────────────
= 净利润 → 净利率 = 净利/营收
```

#### 🔑 关键分析点

| 分析维度 | 关注点 | 健康信号 | 警惕信号 |
|----------|--------|----------|----------|
| 收入质量 | 营收增速 | 稳定增长 | 大幅波动 |
| 毛利率 | 行业对比 | 高于同行 | 持续下降 |
| 费用控制 | 费用率 | 稳定 | 快速上升 |
| 利润来源 | 主营占比 | >90% | 投资收益占比高 |
| 净利润 | 现金含量 | 经营现金流>净利 | 净利高但无现金 |

#### 📊 实践任务

1. **案例分析：中际旭创利润表**
   - 分析营收增长驱动因素
   - 分析毛利率变化原因
   - 评估利润质量

2. **杜邦分析实践**
   - 分解ROE = 净利率 × 资产周转率 × 权益乘数
   - 对比同行业公司ROE差异来源

## Day 3：现金流量表

### 学习材料

#### 📚 必读教材
1. **《财务智慧》- Karen Berman**
   - 章节：现金流量表
   - 时长：约1.5小时

2. **《一本书读懂财报》- 肖星**
   - 章节：现金流量表
   - 时长：约1小时

3. **《财报就像一本故事书》- 刘顺仁**
   - 章节：现金流分析
   - 时长：约2小时

#### 🎥 推荐视频
1. **清华公开课《财务分析与决策》- 肖星**
   - 第6讲：现金流量表
   - 时长：约45分钟

#### 📝 核心概念

```
现金流量表结构
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

┌─────────────────────────────────────┐
│  经营活动现金流（最重要！）          │
├─────────────────────────────────────┤
│  销售商品收到现金                    │
│  - 购买商品支付现金                  │
│  - 支付职工薪酬                      │
│  - 支付税费                          │
│  = 经营活动净现金流                  │
└─────────────────────────────────────┘

┌─────────────────────────────────────┐
│  投资活动现金流                      │
├─────────────────────────────────────┤
│  购建固定资产支付                    │
│  对外投资支付                        │
│  = 投资活动净现金流（通常为负）      │
└─────────────────────────────────────┘

┌─────────────────────────────────────┐
│  筹资活动现金流                      │
├─────────────────────────────────────┤
│  吸收投资收到                        │
│  借款收到                            │
│  - 偿还债务支付                      │
│  - 分红支付                          │
│  = 筹资活动净现金流                  │
└─────────────────────────────────────┘
```

#### 🔑 关键分析点

| 分析维度 | 关注点 | 健康信号 | 警惕信号 |
|----------|--------|----------|----------|
| 经营现金流 | 净现比 | >1 | <0.5或负数 |
| 利润质量 | 经营现金流/净利润 | >1 | <0.5 |
| 投资强度 | 资本支出/折旧 | 合理扩张 | 过度投资 |
| 分红能力 | 自由现金流 | 正数 | 负数 |
| 资金压力 | 现金余额变化 | 充裕 | 持续减少 |

#### 📊 实践任务

1. **现金流质量分析**
   - 对比净利润与经营现金流
   - 分析差异原因
   - 判断利润质量

2. **自由现金流计算**
   ```
   自由现金流 = 经营现金流 - 资本支出
   ```
   - 分析公司是否创造价值
   - 对比同行业FCF表现
