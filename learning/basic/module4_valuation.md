## 一、估值的重要性

### 为什么估值很重要？

```
"价格是你付出的，价值是你得到的"
                —— 巴菲特

估值决定买卖时机：
├─ 价格 < 价值 → 买入机会
├─ 价格 ≈ 价值 → 持有观望
└─ 价格 > 价值 → 卖出/回避

估值方法分类：
├─ 相对估值法：与他人比较
│   └─ PE、PB、PS、PEG、EV/EBITDA
│
└─ 绝对估值法：内在价值计算
    └─ DCF（现金流折现）
```

## 二、相对估值法

### 1. PE（市盈率）

```
PE = 股价 / 每股收益
   = 市值 / 净利润

含义：回本年限
├─ PE=20 → 20年回本
└─ 越低越便宜

判断标准（行业差异大）：
├─ < 15：低估
├─ 15-25：合理
├─ 25-40：偏高
└─ > 40：高估

适用行业：
├─ ✅ 盈利稳定的公司
├─ ✅ 消费、医药、公用事业
└─ ❌ 周期股、亏损公司、高增长公司

PE陷阱：
├─ 低PE可能是陷阱
│   └─ 盈利即将下滑
├─ 高PE可能是机会
│   └─ 高增长支撑
└─ 必须结合增长率看PEG
```

### 2. PEG（市盈率相对盈利增长比率）

```
PEG = PE / 盈利增长率

含义：考虑增长速度的估值

判断标准：
├─ < 1：低估 ✅
├─ = 1：合理
├─ 1-2：偏高
└─ > 2：高估 ⚠️

示例：
┌────────────────────────────────────────┐
│  公司A：PE=30，增长率30%               │
│  PEG = 30/30 = 1.0 → 合理              │
│                                        │
│  公司B：PE=30，增长率15%               │
│  PEG = 30/15 = 2.0 → 偏贵              │
│                                        │
│  同样PE=30，公司A更便宜                │
└────────────────────────────────────────┘

适用：
└─ ✅ 成长股估值
```

### 3. PB（市净率）

```
PB = 股价 / 每股净资产
   = 市值 / 净资产

含义：买入价格是净资产的几倍

判断标准：
├─ < 1：破净，可能低估
├─ 1-2：合理
├─ 2-3：偏高
└─ > 3：高估

适用行业：
├─ ✅ 银行、保险、地产（重资产）
├─ ✅ 周期股底部
└─ ❌ 轻资产、科技、服务

PB陷阱：
├─ 低PB ≠ 便宜
│   └─ 净资产可能虚高
└─ 高PB ≠ 贵
    └─ 资产效率高（ROE高）

PB与ROE关系：
├─ ROE高 → PB可以高
├─ ROE低 → PB应该低
└─ 公式：合理PB ≈ ROE × 市场要求回报率倒数
```

### 4. PS（市销率）

```
PS = 股价 / 每股销售额
   = 市值 / 营业收入

含义：每元销售额的价格

判断标准（行业差异大）：
├─ < 1：低估
├─ 1-3：合理
├─ 3-5：偏高
└─ > 5：高估

适用行业：
├─ ✅ 亏损公司（无PE）
├─ ✅ SaaS、互联网
├─ ✅ 高增长未盈利公司
└─ ❌ 盈利稳定公司（用PE更好）

PS优势：
└─ 销售额比利润更难操纵
```

### 5. EV/EBITDA

```
EV（企业价值）= 市值 + 净债务
             = 市值 + 有息负债 - 现金

EBITDA = 息税折旧摊销前利润
       = 净利润 + 利息 + 税 + 折旧 + 摊销

EV/EBITDA 含义：
└─ 买下整个企业需要多少年回本

判断标准：
├─ < 8：低估
├─ 8-12：合理
├─ 12-15：偏高
└─ > 15：高估

优势：
├─ ✅ 不受资本结构影响
├─ ✅ 不受折旧政策影响
├─ ✅ 适合跨公司比较
└─ ✅ 适合重资产行业

适用：
├─ ✅ 资本密集型行业
├─ ✅ 并购估值
└─ ✅ 跨国公司比较
```

## 三、绝对估值法

### DCF（现金流折现模型）

#### 基本原理

```
企业价值 = 未来自由现金流的折现值之和

公式：
V = Σ FCFt / (1+r)^t

其中：
├─ FCFt = 第t年的自由现金流
├─ r = 折现率（WACC）
└─ t = 年份

核心思想：
├─ 今天的1元 > 明天的1元
├─ 未来现金流需要折现到今天
└─ 折现率反映资金时间价值和风险
```

#### 计算步骤

```
Step 1: 预测未来自由现金流

自由现金流（FCF）= 经营现金流 - 资本支出

预测方法：
├─ 预测期：通常5-10年
├─ 假设：营收增长、毛利率、费用率
└─ 计算：每年FCF

────────────────────────────────────────

Step 2: 计算终值

终值 = 第n年FCF × (1+永续增长率) / (折现率-永续增长率)

永续增长率：
├─ 通常取2-3%
└─ 不超过GDP增速

────────────────────────────────────────

Step 3: 确定折现率（WACC）

WACC = 股权比例 × 股权成本 + 债权比例 × 债权成本 × (1-税率)

股权成本 = 无风险利率 + β × 股权风险溢价

参数选择：
├─ 无风险利率：10年期国债收益率（约3%）
├─ β：行业平均（光模块约1.2）
├─ 股权风险溢价：4-6%
└─ 税率：25%

示例WACC：
└─ 3% + 1.2 × 5% = 9%

────────────────────────────────────────

Step 4: 折现求和

企业价值 = Σ(FCFt/(1+r)^t) + 终值/(1+r)^n

股权价值 = 企业价值 - 净债务
每股价值 = 股权价值 / 总股本
```

#### DCF简化应用

```
简化DCF公式：

合理PE = 1 / (要求回报率 - 永续增长率)

示例：
├─ 要求回报率：10%
├─ 永续增长率：5%
└─ 合理PE = 1/(10%-5%) = 20倍

快速判断：
├─ 稳定增长公司 → PE ≈ 1/(r-g)
└─ 零增长公司 → PE ≈ 1/r
```

#### DCF优缺点

```
优点：
├─ 理论上最正确
├─ 反映内在价值
└─ 考虑全生命周期

缺点：
├─ 预测假设多
├─ 对假设敏感
│   └─ 折现率、增长率微小变化→估值大变化
├─ 终值占比大（通常60-80%）
└─ 主观性强

实战建议：
├─ 做敏感性分析
├─ 多种假设场景
├─ 结合相对估值
└─ 留足安全边际
```

## 四、估值方法选择

### 不同行业适用方法

```
┌─────────────────────────────────────────────┐
│  行业类型        适用方法        原因       │
├─────────────────────────────────────────────┤
│  消费/医药       PE、PEG         盈利稳定   │
│  银行/保险       PB              重资产     │
│  科技/SaaS       PS、PEG         高增长     │
│  周期股          PB（底部）      盈利波动   │
│  制造业          EV/EBITDA       重资产     │
│  公用事业        DCF             现金流稳定 │
└─────────────────────────────────────────────┘
```

### 不同阶段适用方法

```
公司生命周期与估值方法：

导入期：PS（未盈利）
成长期：PEG、PS
成熟期：PE、DCF
衰退期：PB、清算价值
```

## 五、估值实战框架

### 估值流程

```
Step 1: 选择估值方法
├─ 盈利稳定？→ PE
├─ 高增长？→ PEG
├─ 亏损？→ PS
├─ 重资产？→ PB、EV/EBITDA
└─ 现金流稳定？→ DCF

Step 2: 收集数据
├─ 财务数据
├─ 同行数据
└─ 历史数据

Step 3: 计算估值
├─ 相对估值：与历史、同行比较
└─ 绝对估值：DCF计算

Step 4: 敏感性分析
├─ 改变关键假设
├─ 观察估值变化
└─ 得出估值区间

Step 5: 结合判断
├─ 价格 vs 价值
├─ 安全边际
└─ 买卖决策
```

### 估值区间判断

```
估值不是精确数字，而是区间：

估值区间示例：
┌──────────────────────────────────────┐
│  估值区间       判断        操作     │
├──────────────────────────────────────┤
│  低估区间       明显低估    买入 ✅   │
│  合理区间       价格合理    持有/观望 │
│  高估区间       明显高估    卖出/回避 │
└──────────────────────────────────────┘

安全边际：
├─ 巴菲特：买入价格 < 内在价值的2/3
└─ 留出容错空间
```

## 六、估值陷阱

### 常见估值陷阱

```
陷阱1：低PE陷阱
├─ 低PE可能因盈利即将下滑
├─ 周期股顶部常低PE
└─ 需要看盈利趋势

陷阱2：高增长陷阱
├─ 高增长不可持续
├─ PEG计算用历史增速
└─ 需要看增长可持续性

陷阱3：破净陷阱
├─ 低PB可能资产质量差
├─ 资产可能虚高
└─ 需要看资产质量

陷阱4：历史比较陷阱
├─ 历史估值不适用
├─ 基本面已变化
└─ 需要看当前基本面

陷阱5：单一指标陷阱
├─ 只看一个指标
├─ 忽略其他因素
└─ 需要多指标结合
```

### 避免陷阱的方法

```
1. 多指标结合
   └─ PE + PEG + PB 综合判断

2. 历史与同行对比
   └─ 双重验证

3. 关注基本面
   └─ 估值只是参考

4. 敏感性分析
   └─ 多种假设场景

5. 留安全边际
   └─ 不追求精确
```

## 七、估值案例：中际旭创

### 相对估值

```
中际旭创估值数据：
├─ PE(TTM)：约 35倍
├─ PEG：35/50 = 0.7 ✅ 低估
├─ PB：约 5倍
└─ PS：约 5倍

与同行对比：
┌────────────────────────────────────┐
│  公司      PE      PEG     判断   │
├────────────────────────────────────┤
│  中际旭创  35      0.7     合理偏低│
│  新易盛    30      0.8     合理    │
│  天孚通信  60      1.5     偏高    │
└────────────────────────────────────┘

结论：
├─ PE 35倍在成长股中合理
├─ PEG 0.7 说明估值偏低 ✅
└─ 综合判断：估值合理略偏低
```

### 绝对估值（简化DCF）

```
假设：
├─ 未来5年利润增速：30%
├─ 永续增长率：5%
├─ 折现率：10%

简化计算：
合理PE ≈ 1/(10%-5%) × 增长调整

考虑高增长：
├─ 前5年30%增速
├─ 5年后5%永续
└─ 合理PE ≈ 30-40倍

当前PE 35倍 → 在合理区间
```

## 八、学习总结

### 估值方法汇总

```
相对估值法：
├─ PE：盈利稳定公司
├─ PEG：成长股（考虑增长）
├─ PB：重资产公司
├─ PS：亏损/高增长公司
└─ EV/EBITDA：跨公司比较

绝对估值法：
└─ DCF：理论上最正确，但主观性强
```

### 核心要点记忆

```
✅ 估值是艺术不是科学
✅ 估值是区间不是精确数字
✅ 多指标结合更可靠
✅ 相对估值 vs 绝对估值结合
✅ PE看盈利，PEG看增长
✅ PB看资产，PS看销售
✅ DCF看未来现金流
✅ 安全是第一原则：安全边际
✅ 低估不等于马上涨
✅ 高估不等于马上跌
```

### 估值口诀

```
盈利稳定看PE，高增长看PEG
重资产看PB，亏损公司看PS
跨公司比用EV，稳定现金流DCF
历史同行两对比，多种方法综合看
安全边际不能忘，估值区间记心间
```
