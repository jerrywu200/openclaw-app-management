## 二、盈利能力指标详解

### 1. 毛利率

```
毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%

含义：产品定价能力和成本控制能力

行业对比：
┌───────────────────────────────────────┐
│  行业         毛利率范围    特点       │
├───────────────────────────────────────┤
│  白酒         70-90%       品牌溢价   │
│  软件/SaaS    60-80%       低边际成本 │
│  医药         50-80%       研发投入   │
│  半导体       40-60%       技术壁垒   │
│  消费电子     15-30%       竞争激烈   │
│  制造业       15-25%       成本敏感   │
│  零售         10-25%       薄利多销   │
│  汽车         10-20%       价格战     │
└───────────────────────────────────────┘

分析要点：
✅ 毛利率稳定或上升 = 竞争力强
⚠️ 毛利率下降 = 竞争加剧/成本上升
❌ 毛利率大幅波动 = 经营不稳定
```

### 2. 净利率

```
净利率 = 净利润 / 营业收入 × 100%

含义：最终盈利能力

净利率 = 毛利率 - 费用率 - 税率

影响净利率的因素：
├── 毛利率（最核心）
├── 销售费用率
├── 管理费用率
├── 研发费用率
├── 财务费用率
└── 所得税率

净利率健康标准：
├─ > 20%：优秀（茅台、医药）
├─ 10-20%：良好（消费、科技）
├─ 5-10%：一般（制造业）
└─ < 5%：偏低（需改善）
```

### 3. ROE（净资产收益率）

```
ROE = 净利润 / 净资产 × 100%

杜邦分解：ROE = 净利率 × 资产周转率 × 权益乘数

ROE判断标准（巴菲特标准）：
├─ > 20%：优秀
├─ 15-20%：良好
├─ 10-15%：一般
└─ < 10%：较差

连续高ROE的公司特征：
├── 有护城河（品牌/技术/规模）
├── 轻资产或高效资产
├── 合理杠杆
└── 稳定的盈利能力

ROE质量判断：
├─ 净利率驱动 > 周转率驱动 > 杠杆驱动
└─ 高质量ROE = 高净利率 + 适度周转 + 适度杠杆
```

### 4. ROA（总资产收益率）

```
ROA = 净利润 / 总资产 × 100%

含义：资产利用效率

ROA = 净利率 × 资产周转率

ROA vs ROE：
├─ ROA不受杠杆影响
├─ ROA更适合跨公司比较
└─ ROA低但ROE高 = 高杠杆

ROA判断标准：
├─ > 10%：优秀
├─ 5-10%：良好
├─ 2-5%：一般
└─ < 2%：较差
```

### 5. ROIC（投入资本回报率）

```
ROIC = 税后经营利润 / 投入资本 × 100%

投入资本 = 有息负债 + 股东权益 - 现金

含义：真正的经营效率
├─ 剔除财务杠杆影响
├─ 剔除现金影响
└─ 反映核心业务回报率

ROIC判断标准：
├─ > 15%：优秀
├─ 10-15%：良好
├─ 5-10%：一般
└─ < 5%：价值毁灭

ROIC > WACC（加权资本成本）= 创造价值
ROIC < WACC = 毁灭价值
```

## 三、偿债能力指标详解

### 1. 流动比率

```
流动比率 = 流动资产 / 流动负债

含义：短期偿债能力

判断标准：
├─ > 2：很强
├─ 1.5-2：健康
├─ 1-1.5：需关注
└─ < 1：危险

注意：
├─ 过高也不好（资金闲置）
├─ 行业差异大
└─ 需结合行业判断
```

### 2. 速动比率

```
速动比率 = (流动资产 - 存货) / 流动负债

含义：更严格的短期偿债能力（剔除变现慢的存货）

判断标准：
├─ > 1：健康
├─ 0.8-1：正常
└─ < 0.8：需关注

适用：
└── 存货占比高的公司必须看速动比率
```

### 3. 资产负债率

```
资产负债率 = 总负债 / 总资产 × 100%

含义：财务杠杆程度

判断标准：
├─ < 40%：保守，财务稳健
├─ 40-60%：适中，合理杠杆
├─ 60-70%：偏高，需关注
└─ > 70%：高风险

行业差异：
├─ 银行：90%+（正常）
├─ 地产：70-80%（偏高）
├─ 制造业：40-60%
└─ 科技：20-40%
```

### 4. 利息保障倍数

```
利息保障倍数 = EBIT / 利息费用
            = (利润总额 + 利息费用) / 利息费用

含义：偿付利息的能力

判断标准：
├─ > 5：很强
├─ 3-5：安全
├─ 1.5-3：需关注
└─ < 1.5：危险

< 1 = 利润不够还利息 = 严重问题
```

### 5. 现金比率

```
现金比率 = (货币资金 + 交易性金融资产) / 流动负债

含义：最严格的偿债能力（只能用现金还债）

判断标准：
├─ > 1：非常安全
├─ 0.5-1：安全
└─ < 0.5：需关注
```

## 四、运营效率指标详解

### 1. 存货周转率

```
存货周转率 = 营业成本 / 平均存货
存货周转天数 = 365 / 存货周转率

含义：存货变现速度

分析：
├─ 周转率越高越好
├─ 天数越短越好
└─ 行业差异大

判断标准（制造业参考）：
├─ < 30天：优秀
├─ 30-60天：良好
├─ 60-90天：一般
└─ > 90天：需关注

红旗信号：
└─ 周转天数持续增加 = 产品滞销
```

### 2. 应收账款周转率

```
应收账款周转率 = 营业收入 / 平均应收账款
应收账款周转天数 = 365 / 应收账款周转率

含义：回款速度

分析：
├─ 周转率越高越好
├─ 天数越短越好
└─ 反映客户质量和议价能力

判断标准：
├─ < 30天：优秀（预收/现结）
├─ 30-60天：良好
├─ 60-90天：一般
└─ > 90天：回款慢

红旗信号：
└─ 应收账款增速 > 营收增速 = 放宽账期促销
```

### 3. 总资产周转率

```
总资产周转率 = 营业收入 / 平均总资产

含义：资产利用效率

判断标准（行业差异大）：
├─ 零售业：2-3次
├─ 制造业：0.5-1.5次
├─ 重资产行业：0.3-0.8次
└─ 轻资产行业：1-2次

分析：
├─ 越高说明资产效率越高
└─ 结合行业对比
```

### 4. 应付账款周转率

```
应付账款周转率 = 营业成本 / 平均应付账款
应付账款周转天数 = 365 / 应付账款周转率

含义：占用供应商资金能力

分析：
├─ 天数越长 = 越能占用供应商资金
├─ 天数长 = 议价能力强（好事）
└─ 但过长可能影响供应商关系

判断：
├─ 60-90天：正常
├─ > 90天：议价能力强
└─ < 30天：议价能力弱
```

## 五、成长性指标详解

### 1. 营收增长率

```
营收增长率 = (本期营收 - 上期营收) / 上期营收 × 100%

含义：市场规模扩张速度

判断标准：
├─ > 30%：高速增长
├─ 15-30%：快速增长
├─ 5-15%：稳定增长
├─ 0-5%：缓慢增长
└─ < 0：衰退

可持续增长判断：
├─ 连续3年>15% = 成长股特征
└─ 增长来源分析（量增vs价增）
```

### 2. 净利润增长率

```
净利润增长率 = (本期净利 - 上期净利) / 上期净利 × 100%

含义：盈利扩张速度

健康增长特征：
├─ 利润增速 ≥ 营收增速（规模效应）
└─ 增长稳定，非一次性因素

红旗信号：
└─ 利润增速远低于营收增速 = 盈利能力下降
```

### 3. EPS增长率

```
EPS = 净利润 / 总股本
EPS增长率 = (本期EPS - 上期EPS) / 上期EPS × 100%

含义：每股盈利增长

分析：
├─ EPS增长 > 净利润增长 = 股份回购
└─ EPS增长 < 净利润增长 = 增发稀释
```

### 4. 经营现金流增长率

```
经营现金流增长率 = (本期OCF - 上期OCF) / 上期OCF × 100%

含义：造血能力增长

健康判断：
└─ 经营现金流增速与净利润增速匹配
```

## 六、指标综合分析框架

### 财务健康诊断流程

```
Step 1: 盈利能力
├── 毛利率 > 行业平均？
├── 净利率 > 10%？
├── ROE > 15%？
└── ROIC > WACC？

Step 2: 偿债能力
├── 流动比率 > 1.5？
├── 资产负债率 < 60%？
└── 利息保障倍数 > 3？

Step 3: 运营效率
├── 存货周转率稳定？
├── 应收周转率稳定？
└── 总资产周转率合理？

Step 4: 成长性
├── 营收增速 > 15%？
├── 利润增速 ≥ 营收增速？
└── 现金流匹配？

Step 5: 综合判断
└── 各指标交叉验证
```

### 红旗信号汇总

```
盈利能力红旗：
├─ 毛利率持续下降
├─ 净利率 < 5%
├─ ROE < 10%
└─ ROIC < WACC

偿债能力红旗：
├─ 流动比率 < 1
├─ 资产负债率 > 70%
└─ 利息保障倍数 < 1.5

运营效率红旗：
├─ 存货周转天数大幅增加
├─ 应收账款增速 > 营收增速
└─ 应收账款周转天数 > 90天

成长性红旗：
├─ 营收负增长
├─ 利润增速远低于营收增速
└─ 经营现金流与利润背离
```

## 七、学习总结

### 核心指标记忆

```
盈利能力看ROE，杜邦分解找驱动
偿债能力看比率，流动速动负债率
运营效率看周转，存货应收总资产
成长性看增速，营收利润现金流
```

### 分析口诀

```
毛利率高产品强，净利率高效率棒
ROE高股东喜，分解看谁在帮忙
流动比率看短期，负债率看长期
周转率高资产活，回款快慢看应收
增长稳定最可贵，质量要看现金流
```
