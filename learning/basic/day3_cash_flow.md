## 一、现金流量表的本质

### 为什么现金流量表最重要？

```
利润是"观点"，现金是"事实"

企业倒闭不是因为亏损，而是因为没钱

现金流量表：
记录公司现金的来龙去脉
是最难粉饰的报表
```

### 三大现金流分类

```
┌─────────────────────────────────────────┐
│  经营活动现金流（造血能力）              │
├─────────────────────────────────────────┤
│  销售商品、提供劳务收到的现金            │
│  - 购买商品、接受劳务支付的现金          │
│  - 支付给职工的现金                      │
│  - 支付的各项税费                        │
│  = 经营活动净现金流                      │
└─────────────────────────────────────────┘

┌─────────────────────────────────────────┐
│  投资活动现金流（投资扩张）              │
├─────────────────────────────────────────┤
│  - 购建固定资产、无形资产支付的现金      │
│  - 投资支付的现金                        │
│  + 收回投资收到的现金                    │
│  + 取得投资收益收到的现金                │
│  = 投资活动净现金流（通常为负）          │
└─────────────────────────────────────────┘

┌─────────────────────────────────────────┐
│  筹资活动现金流（融资能力）              │
├─────────────────────────────────────────┤
│  + 吸收投资收到的现金                    │
│  + 取得借款收到的现金                    │
│  - 偿还债务支付的现金                    │
│  - 分配股利、利润支付的现金              │
│  = 筹资活动净现金流                      │
└─────────────────────────────────────────┘
```

## 二、经营现金流分析

### 经营现金流与净利润对比

```
净现比 = 经营现金流净额 / 净利润

> 1：利润质量高 ✅
≈ 1：利润质量正常 ✅
< 0.5：利润质量低 ⚠️
< 0：利润可能是纸面富贵 ❌

差异原因分析：
经营现金流 > 净利润：
├── 应收账款减少（回款好）
├── 预收款增加（预收多）
├── 存货减少（周转快）
└── 折旧摊销大（非现金支出）

经营现金流 < 净利润：
├── 应收账款增加（赊销多）
├── 存货增加（积压）
└── 预付款增加（议价弱）
```

### 经营现金流健康判断

```
经营现金流判断标准：

持续为正 ✅ → 业务能自我造血
持续为负 ❌ → 业务失血，需融资
忽正忽负 ⚠️ → 经营不稳定

增长趋势：
与净利润同步增长 ✅
高于净利润增长 ✅✅
低于净利润增长 ⚠️
```

## 三、投资现金流分析

### 投资现金流解读

```
投资现金流通常为负（花钱投资）

- 资本支出（CAPEX）
  = 购建固定资产、无形资产支付的现金

分析要点：
├── 投资是否必要
├── 投资回报如何
└── 投资强度是否合理

资本支出/折旧：
≈ 1：维持性投资（保持现状）
> 1：扩张性投资（在增长）
< 1：收缩性投资（在萎缩）
```

### 投资现金流红旗信号

```
⚠️ 投资支付现金大幅增加
    → 可能在盲目扩张

⚠️ 投资活动流入占比高
    → 可能变卖资产维持

⚠️ 资本支出持续高于同行
    → 投资效率可能低下
```

## 四、筹资现金流分析

### 筹资现金流解读

```
筹资活动反映融资能力

正面信号：
├── 股权融资（增发）
├── 债务融资合理增长
└── 分红稳定

负面信号：
├── 持续借新还旧
├── 借款利率上升
├── 分红能力下降
└── 大股东减持
```

## 五、自由现金流

### 自由现金流计算

```
自由现金流（FCF）
= 经营现金流 - 资本支出

含义：企业可自由支配的现金

FCF > 0：
✅ 企业能自我维持
✅ 有分红/回购能力
✅ 有扩张能力

FCF < 0：
⚠️ 需要外部融资
⚠️ 无法分红
⚠️ 可能发展受限
```

### 自由现金流与估值

```
DCF估值法的核心：
企业价值 = 未来自由现金流的折现值

自由现金流折现模型：
1. 预测未来5-10年FCF
2. 计算终值
3. 折现到今天
4. 加总得到企业价值

简单判断：
FCF稳定增长的公司 = 高质量公司
```

## 六、现金流组合分析

### 企业生命周期判断

```
            经营  投资  筹资
初创期       -     -     +
成长期       +     -     +
成熟期       +     -     -
衰退期       -     +     -

"+" 表示净流入，"-" 表示净流出

成熟期企业特征：
├── 经营现金流充沛 ✅
├── 投资活动流出（维护性）
├── 筹资活动流出（还债+分红）
└── 典型：茅台、格力
```

### 现金流组合诊断

| 经营 | 投资 | 筹资 | 诊断 |
|------|------|------|------|
| + | - | - | ✅ 成熟健康型 |
| + | - | + | 🔄 成长扩张型 |
| + | + | - | ⚠️ 收缩型（卖资产） |
| - | - | + | ⚠️ 初创/困难型 |
| - | + | + | ❌ 靠融资+卖资产 |
| - | - | - | ❌ 危机型 |

## 八、三大报表勾稽关系

```
资产负债表 ← 利润表
     ↑            ↓
     │       净利润
     │            ↓
现金流量表 ← 经营现金流

核心勾稽：
1. 资产负债表期末 = 期初 + 本期变动
2. 净利润 → 留存收益 → 所有者权益
3. 现金流净额 = 货币资金变动
4. 经营现金流 vs 净利润 → 利润质量
```

### 综合分析框架

```
1. 看资产（资产负债表）
   └── 家底如何

2. 看利润（利润表）
   └── 赚钱能力

3. 看现金（现金流量表）
   └── 真金白银

4. 三表交叉验证
   ├── 利润质量（现金流匹配）
   ├── 资产效率（周转率）
   └── 增长可持续性
```
