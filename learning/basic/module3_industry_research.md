## 一、行业研究的重要性

### 为什么行业研究很重要？

```
"选对赛道，成功一半"

行业决定天花板：
├─ 行业空间 = 公司成长上限
├─ 行业格局 = 公司盈利能力
├─ 行业周期 = 公司业绩波动
└─ 行业趋势 = 公司未来方向

好行业 + 好公司 = 优秀投资
好行业 + 差公司 = 可能反转
差行业 + 好公司 = 很难突破
差行业 + 差公司 = 避免投资
```

## 二、行业生命周期分析

### 四个阶段

```
┌─────────────────────────────────────────────────┐
│                                                 │
│  营收                                           │
│    ↑                                            │
│    │           ┌──────┐                         │
│    │          /        \                        │
│    │         /  成熟期  \                       │
│    │        /            \                      │
│    │   ┌───┐              \───────┐            │
│    │  / 成长 \                     衰退期      │
│    │ /        \                               │
│    │/  导入期   \                             │
│    └──────────────────────────────────→ 时间   │
│                                                 │
└─────────────────────────────────────────────────┘
```

### 各阶段特征

| 阶段 | 特征 | 增速 | 竞争 | 投资策略 |
|------|------|------|------|----------|
| **导入期** | 市场培育 | 不稳定 | 少 | 风险投资 |
| **成长期** | 快速扩张 | >20% | 增加 | 最佳投资期 ✅ |
| **成熟期** | 稳定增长 | 5-15% | 激烈 | 选择龙头 |
| **衰退期** | 市场萎缩 | <0% | 退出 | 回避 ⚠️ |

### 判断方法

```
判断行业处于哪个阶段：

1. 看增速
   ├─ >30%：成长期
   ├─ 10-30%：成长后期/成熟前期
   ├─ 0-10%：成熟期
   └─ <0%：衰退期

2. 看渗透率
   ├─ <10%：导入期
   ├─ 10-50%：成长期
   ├─ 50-80%：成熟期
   └─ >80%：衰退期

3. 看竞争格局
   ├─ 玩家少：导入期/成长前期
   ├─ 玩家多：成长后期/成熟期
   └─ 玩家退出：衰退期

4. 看产品生命周期
   ├─ 新产品涌现：成长期
   ├─ 产品同质化：成熟期
   └─ 产品被替代：衰退期
```

## 三、行业竞争格局分析

### 波特五力模型

```
          ┌─────────────────────┐
          │   潜在进入者威胁     │
          │  （进入壁垒分析）    │
          └──────────┬──────────┘
                     │
                     ▼
┌──────────┐    ┌─────────────────────┐    ┌──────────┐
│          │    │                     │    │          │
│  供应商   │───▶│    行业竞争格局     │◀───│  购买者   │
│  议价能力 │    │   （现有竞争者）    │    │  议价能力 │
│          │    │                     │    │          │
└──────────┘    └──────────┬──────────┘    └──────────┘
                     │
                     ▼
          ┌─────────────────────┐
          │    替代品威胁        │
          │  （替代品分析）      │
          └─────────────────────┘
```

### 五力详细分析

#### 1️⃣ 现有竞争者竞争程度

```
影响因素：
├─ 行业增长率
│   └─ 高增长 = 竞争缓和
│   └─ 低增长 = 竞争激烈
│
├─ 行业集中度
│   └─ 高集中（寡头）= 竞争缓和
│   └─ 低集中（分散）= 竞争激烈
│
├─ 产品差异化
│   └─ 差异化强 = 竞争缓和
│   └─ 同质化 = 价格战
│
├─ 退出壁垒
│   └─ 退出容易 = 竞争缓和
│   └─ 退出困难 = 恶性竞争
│
└─ 固定成本占比
    └─ 高固定成本 = 产能过剩风险
```

#### 2️⃣ 潜在进入者威胁

```
进入壁垒分析：

高壁垒（护城河强）：
├─ 资金壁垒：需要巨额投资
├─ 技术壁垒：专利、know-how
├─ 品牌壁垒：消费者认知
├─ 渠道壁垒：销售网络
├─ 规模壁垒：规模经济
├─ 政策壁垒：牌照、准入
└─ 网络效应：用户粘性

低壁垒（竞争激烈）：
├─ 资金需求少
├─ 技术门槛低
├─ 品牌效应弱
└─ 政策限制少
```

#### 3️⃣ 替代品威胁

```
替代品类型：
├─ 直接替代：同类产品
│   └─ 例：燃油车 vs 电动车
│
├─ 间接替代：不同功能但满足相同需求
│   └─ 例：高铁 vs 航空
│
└─ 颠覆性替代：新技术革命
    └─ 例：数码相机 vs 胶卷

判断标准：
├─ 替代品性价比 > 本行业 → 高威胁
├─ 替代品性价比 ≈ 本行业 → 中等威胁
└─ 替代品性价比 < 本行业 → 低威胁
```

#### 4️⃣ 供应商议价能力

```
供应商强势信号：
├─ 供应商集中度高
├─ 替代投入品少
├─ 买方分散
├─ 投入品差异化强
└─ 前向整合威胁大

供应商弱势信号：
├─ 供应商分散
├─ 替代品多
├─ 买方集中
└─ 投入品标准化
```

#### 5️⃣ 购买者议价能力

```
买方强势信号：
├─ 买方集中度高
├─ 产品同质化
├─ 转换成本低
├─ 买方后向整合威胁
└─ 价格敏感度高

买方弱势信号：
├─ 买方分散
├─ 产品差异化强
├─ 转换成本高
└─ 品牌效应强
```

## 四、产业链分析

### 产业链位置分析

```
上游 → 中游 → 下游

┌─────────┐   ┌─────────┐   ┌─────────┐
│  原材料  │──▶│  制造   │──▶│  销售   │
│  供应商  │   │  加工   │   │  消费者  │
└─────────┘   └─────────┘   └─────────┘

产业链位置决定：
├─ 议价能力
├─ 盈利能力
└─ 受周期影响程度
```

### 各环节特点

| 环节 | 特点 | 盈利稳定性 | 典型行业 |
|------|------|------------|----------|
| **上游** | 资源垄断 | 波动大 | 矿产、能源 |
| **中游** | 竞争激烈 | 波动中 | 制造、加工 |
| **下游** | 接触终端 | 波动小 | 消费、零售 |

### 产业链分析框架

```
分析要点：

1. 产业链结构
   └─ 各环节有哪些玩家

2. 价值分布
   └─ 利润在哪个环节

3. 议价能力
   └─ 谁对价格有主导权

4. 瓶颈环节
   └─ 哪个环节制约产业发展

5. 变化趋势
   └─ 产业链如何演变
```

## 五、商业模式分析

### 商业模式三要素

```
┌─────────────────────────────────────────┐
│           商业模式画布                   │
├─────────────────────────────────────────┤
│                                         │
│  1. 创造什么价值？（价值主张）          │
│     └─ 解决什么痛点？                  │
│                                         │
│  2. 如何创造价值？（核心流程）          │
│     └─ 核心竞争力是什么？              │
│                                         │
│  3. 如何获取价值？（盈利模式）          │
│     └─ 怎么赚钱？                      │
│                                         │
└─────────────────────────────────────────┘
```

### 常见商业模式

```
1. 产品销售模式
   └─ 卖产品赚钱（苹果、茅台）

2. 服务收费模式
   └─ 卖服务赚钱（咨询、SaaS）

3. 平台抽佣模式
   └─ 抽佣金赚钱（淘宝、美团）

4. 广告变现模式
   └─ 卖流量赚钱（Google、抖音）

5. 金融杠杆模式
   └─ 利差赚钱（银行、保险）

6. 会员订阅模式
   └─ 会费赚钱（Netflix、Costco）
```

### 商业模式评估

```
好商业模式特征：
├─ 可复制：能规模化
├─ 高粘性：客户难离开
├─ 高毛利：盈利空间大
├─ 现金流好：先收钱后服务
├─ 轻资产：ROE高
└─ 护城河：难以被模仿

差商业模式特征：
├─ 难复制：依赖个人/资源
├─ 低粘性：客户易流失
├─ 低毛利：价格战
├─ 现金流差：先垫资后收款
├─ 重资产：ROE低
└─ 无壁垒：易被替代
```

## 六、行业分析框架总结

### 分析流程

```
Step 1: 行业定义
├─ 行业边界
├─ 主要产品
└─ 目标客户

Step 2: 行业空间
├─ 市场规模
├─ 增长速度
└─ 驱动因素

Step 3: 生命周期
├─ 当前阶段
├─ 持续时间
└─ 未来趋势

Step 4: 竞争格局
├─ 波特五力
├─ 主要玩家
└─ 集中度

Step 5: 产业链
├─ 位置分析
├─ 议价能力
└─ 价值分布

Step 6: 商业模式
├─ 盈利模式
├─ 核心竞争力
└─ 护城河

Step 7: 政策环境
├─ 监管政策
├─ 扶持政策
└─ 风险因素

Step 8: 投资判断
├─ 好行业/差行业
├─ 好赛道/差赛道
└─ 关注要点
```

## 八、学习总结

### 核心方法论

```
行业研究三步法：

1. 定性分析
   ├── 行业定义
   ├── 生命周期
   └── 竞争格局（五力模型）

2. 定量分析
   ├── 市场空间
   ├── 增长速度
   └── 盈利水平

3. 判断结论
   ├── 好行业/差行业
   ├── 机会/风险
   └─ 关注要点
```

### 核心要点记忆

```
✅ 行业决定天花板
✅ 生命周期看增速和渗透率
✅ 波特五力判断竞争格局
✅ 产业链位置决定议价能力
✅ 商业模式决定盈利质量
✅ 护城河比规模更重要
✅ 政策影响不容忽视
✅ 好行业+好公司=好投资
```
