## 三、技术分析三种支撑位判断方法

### 方法1：震荡区间法

```
原理：价格在区间内震荡，下沿形成支撑

步骤：
1. 找出近期震荡的K线
2. 确定区间的最高价和最低价
3. 区间下沿 = 震荡期间多次测试的低点

本例：
├─ 震荡期间：3/18-3/19
├─ 区间范围：569-592元
└─ 下沿支撑：约580元
```

### 方法2：平台支撑法

```
原理：横盘整理期间筹码密集交换，形成支撑

步骤：
1. 找出横盘整理的区间
2. 计算平台中枢（中位数或平均价）
3. 中枢位置形成支撑

本例：
├─ 平台期间：3/10-3/17
├─ 平台区间：525-570元
├─ 平台中位：约547-550元
└─ 支撑位：约550元
```

### 方法3：历史低点法

```
原理：多次测试的低点形成支撑带

步骤：
1. 找出近期所有重要低点
2. 观察哪些价位被多次测试
3. 多次测试的价位形成支撑

本例：
├─ 低点1：506元（3/9恐慌低点）
├─ 低点2：522元（2/24）
├─ 低点3：550元（2/9、3/18）
└─ 支撑带：520-550元
```

## 五、总结

### 支撑位计算公式

```
支撑位 = min(震荡下沿, 平台中枢, 历史低点)

具体计算：
├─ 震荡下沿 = min(震荡期最低价) ≈ 570元
├─ 平台中位 = (平台最高+平台最低)/2 ≈ 550元
├─ 历史低点 = 近期多次测试低点 ≈ 520-550元
└─ 综合判断：550-580元
```
