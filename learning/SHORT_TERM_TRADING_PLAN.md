## 二、学习模块设计

### 模块14：分时图与盘口分析（核心）

**学习目标**：掌握分时图解读、盘口动态、日内节奏

#### 14.1 分时图基础
```
分时图要素：
├─ 白线：当前价格
├─ 黄线：均价线（重要支撑/压力）
├─ 红绿柱：成交量
├─ 涨跌幅：即时涨跌
└─ 委买卖盘：五档挂单

关键形态：
1. 黄白线关系
   - 白线在黄线上方：强势
   - 白线在黄线下方：弱势
   - 黄白线粘合：震荡

2. 分时走势形态
   - 阶梯式上涨：健康上涨
   - 直线拉升：诱多/真拉升需验证
   - 震荡下行：弱势
   - V型反转：情绪转折

3. 量价配合
   - 上涨放量：健康
   - 上涨缩量：乏力
   - 下跌放量：恐慌
   - 下跌缩量：正常
```

#### 14.2 盘口解读
```
五档盘口：
┌─────────────────────────┐
│ 卖5: 100.50  500手      │
│ 卖4: 100.40  300手      │
│ 卖3: 100.30  200手      │
│ 卖2: 100.20  150手      │
│ 卖1: 100.10  100手      │
├─────────────────────────┤
│ 买1: 100.00  200手      │
│ 买2: 99.90   300手      │
│ 买3: 99.80   250手      │
│ 买4: 99.70   180手      │
│ 买5: 99.60   400手      │
└─────────────────────────┘

关键观察：
1. 委比 = (买盘-卖盘)/(买盘+卖盘) × 100%
   - >50%：强势
   - <-50%：弱势

2. 委差 = 买盘总量 - 卖盘总量
   - 正值：买盘强
   - 负值：卖盘强

3. 内外盘
   - 外盘：主动买入（看多）
   - 内盘：主动卖出（看空）
   - 外盘>内盘：多头占优

4. 大单异动
   - 连续大买单：主力吸筹/拉升
   - 连续大卖单：主力出货
   - 托单/压单：诱多/诱空
```

#### 14.3 日内节奏
```
时间节奏（A股）：
├─ 9:15-9:25 集合竞价
│   └─ 观察开盘价位置、量能
├─ 9:30-10:00 开盘半小时
│   └─ 最活跃时段，方向确认
├─ 10:00-11:30 上午盘
│   └─ 震荡或延续趋势
├─ 13:00-14:00 下午开盘
│   └─ 常有方向选择
├─ 14:00-14:30 下午盘
│   └─ 趋势延续或转折
└─ 14:30-15:00 尾盘
    └─ 突袭/回落，次日预期

关键时点：
- 9:25 竞价结束 → 判断开盘强弱
- 9:30-9:35 → 开盘5分钟定调
- 10:00 → 第一小时收盘，确认方向
- 11:30 → 上午收盘，中午休市
- 14:30 → 尾盘开始，关注异动
- 14:57-15:00 → 竞价收盘
```

### 模块15：短线技术指标

**学习目标**：掌握日内/短线常用指标

#### 15.1 均线系统（短线版）
```
短线均线：
├─ MA5：超短线，日内容易触及
├─ MA10：短线支撑/压力
├─ MA20：波段趋势线
└─ MA60：中期趋势

使用规则：
1. 价格在MA5上方 + MA5向上 → 短线多头
2. 价格在MA5下方 + MA5向下 → 短线空头
3. MA5金叉MA10 → 短线买入信号
4. MA5死叉MA10 → 短线卖出信号
```

#### 15.2 MACD短线应用
```
MACD参数：默认(12,26,9)

短线信号：
1. DIF上穿DEA → 金叉买入
2. DIF下穿DEA → 死叉卖出
3. 红柱放大 → 多头增强
4. 绿柱放大 → 空头增强
5. 顶背离 → 卖出信号
6. 底背离 → 买入信号

注意：
- 日线MACD滞后，适合波段
- 30分钟MACD更适合短线
- 15分钟MACD适合日内
```

#### 15.3 KDJ短线指标
```
KDJ参数：默认(9,3,3)

信号解读：
├─ K>80, D>80 → 超买，注意回调
├─ K<20, D<20 → 超卖，注意反弹
├─ K上穿D → 金叉买入
├─ K下穿D → 死叉卖出
└─ J>100 或 J<0 → 极端信号

短线用法：
1. 超买区死叉 → 强烈卖出
2. 超卖区金叉 → 强烈买入
3. 结合趋势使用，避免逆势
```

#### 15.4 RSI相对强弱
```
RSI参数：常用14日或6日

区间含义：
├─ RSI>80 → 严重超买
├─ RSI 70-80 → 超买区
├─ RSI 50-70 → 强势区
├─ RSI 30-50 → 弱势区
├─ RSI 20-30 → 超卖区
└─ RSI<20 → 严重超卖

短线用法：
1. 超买区转头向下 → 卖出
2. 超卖区转头向上 → 买入
3. 背离信号 → 趋势转折
```

#### 15.5 布林带短线应用
```
布林带参数：默认(20,2)

通道含义：
├─ 上轨：压力位
├─ 中轨：趋势线（MA20）
└─ 下轨：支撑位

短线信号：
1. 价格触及上轨 → 超买，可能回调
2. 价格触及下轨 → 超卖，可能反弹
3. 布林收口 → 变盘在即
4. 布林开口 → 趋势启动

策略：
- 突破上轨 + 放量 → 强势追涨
- 跌破下轨 + 放量 → 恐慌观望
- 中轨支撑 → 买入机会
```

### 模块16：短线交易策略

**学习目标**：掌握短线买入/卖出策略

#### 16.1 短线买入信号
```
买入信号组合：

1. 早盘强势买入
   - 高开 + 量比>2
   - 5分钟内站稳分时均线
   - MACD金叉

2. 回调买入
   - 回调至MA5/MA10
   - 量能萎缩
   - KDJ超卖区金叉

3. 突破买入
   - 突破前高/平台
   - 放量确认
   - 布林开口

4. 尾盘潜伏
   - 尾盘拉升
   - 量能异动
   - 次日高开预期
```

#### 16.2 短线卖出信号
```
卖出信号组合：

1. 冲高回落
   - 冲高后跌破分时均线
   - 量能放大
   - 卖盘涌出

2. 技术死叉
   - MA5死叉MA10
   - MACD死叉
   - KDJ死叉在超买区

3. 顶背离
   - 价格新高，指标不新高
   - MACD顶背离
   - 量价背离

4. 止损触发
   - 跌破买入价-3%
   - 跌破MA5
   - 跌破分时均线
```

#### 16.3 T+0策略（日内）
```
T+0前提：
- 已有持仓
- 震荡行情
- 10点前完成

正向T（先买后卖）：
1. 早盘低开/回调
2. 在支撑位买入
3. 反弹至压力位卖出

反向T（先卖后买）：
1. 早盘高开/冲高
2. 在压力位卖出
3. 回调至支撑位买回

注意事项：
- 单次T利润目标1-3%
- 严格执行，不贪心
- 判断失误立即止损
```

### 模块17：短线风险控制

**学习目标**：掌握短线止损止盈纪律

#### 17.1 止损规则
```
短线止损铁律：

1. 固定止损
   - 单笔亏损 ≤ 3%
   - 日内亏损 ≤ 5%
   - 单周亏损 ≤ 10%

2. 技术止损
   - 跌破分时均线
   - 跌破MA5
   - 跌破买入价-3%

3. 时间止损
   - 买入后30分钟无盈利 → 观望
   - 买入后1小时亏损 → 考虑止损
   - 收盘前亏损 → 必须止损

4. 情绪止损
   - 心态失衡 → 停止交易
   - 连续亏损 → 休息复盘
```

#### 17.2 止盈规则
```
短线止盈策略：

1. 固定止盈
   - 盈利3% → 移动止损至成本
   - 盈利5% → 止损移至+2%
   - 盈利8% → 考虑减仓一半

2. 技术止盈
   - 冲高回落 → 部分止盈
   - 触及压力位 → 减仓
   - 顶背离 → 全部止盈

3. 分批止盈
   - 第一目标位：减1/3
   - 第二目标位：减1/2
   - 第三目标位：清仓

4. 尾盘止盈
   - 盈利股尾盘不封板 → 减仓
   - 防止次日低开
```
