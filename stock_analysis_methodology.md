## 一、量价分析流程

### 1. 获取数据
- 近30日K线数据（开盘/收盘/最高/最低/成交量）
- 数据来源：stock_db `get_stock_kline()`

### 2. 统计指标
- 阳天数/阴天数
- 涨跌幅、振幅
- 成交额变化

### 3. 量价配合判断

| 量价关系 | 含义 | 判断 |
|----------|------|------|
| 上涨放量 | 资金买入意愿强 | 健康 ✅ |
| 上涨缩量 | 买盘不足 | 乏力 ⚠️ |
| 下跌放量 | 恐慌抛售 | 危险 ⚠️ |
| 下跌缩量 | 惜售 | 正常 ✅ |

### 4. 健康度评分
- 上涨放量比例
- 下跌缩量比例
- 大阳线放量比例

### 5. 多空判断
- 综合量价配合评分
- 结合K线形态

## 二、财务报表分析框架

### 1. 资产负债表分析

**核心等式**：`资产 = 负债 + 所有者权益`

**关键指标**：
| 指标 | 标准 | 说明 |
|------|------|------|
| 流动比率 | > 1.5 | 偿债能力 |
| 速动比率 | > 1 | 快速偿债能力 |
| 资产负债率 | < 60% | 财务稳健 |
| 应收账款增速 | ≤ 营收增速 | 收入质量 |
| 存货周转率 | 稳定或上升 | 库存管理 |

**红旗信号**：
- 应收账款增速 > 营收增速（可能虚增收入）
- 存货持续积压（滞销风险）
- 商誉占比过高（减值风险）
- 资产负债率持续上升（财务恶化）

### 2. 利润表分析

**核心流程**：`营收 → 毛利 → 营业利润 → 净利润`

**关键指标**：
| 指标 | 含义 | 判断标准 |
|------|------|----------|
| 毛利率 | 产品竞争力 | 稳定或上升为佳 |
| 净利率 | 最终盈利能力 | >10%良好，>20%优秀 |
| 费用率 | 经营效率 | 控制合理 |
| 净现比 | 利润质量 | > 1 为佳 |

**红旗信号**：
- 毛利率异常波动
- 投资收益占比过高
- 营业外收入大增
- 经营现金流 < 净利润

### 3. 现金流量表分析

**三大现金流**：经营/投资/筹资

**核心判断**：
| 指标 | 标准 | 说明 |
|------|------|------|
| 经营现金流 | 持续为正 | 主业造血能力 |
| 净现比 | > 1 | 利润质量高 |
| 自由现金流 | > 0 | 现金创造能力 |

**公式**：`自由现金流 = 经营现金流 - 资本支出`

**红旗信号**：
- 经营现金流持续为负
- 自由现金流持续为负
- 靠融资+卖资产维持

### 4. 杜邦分析

**公式**：`ROE = 净利率 × 资产周转率 × 权益乘数`

**三种模式**：
| 模式 | 代表公司 | 特点 |
|------|----------|------|
| 高净利模式 | 茅台 | 品牌溢价 |
| 高周转模式 | 沃尔玛 | 薄利多销 |
| 高杠杆模式 | 银行 | 借钱生钱 |

## 三、财务指标分析框架

### 1. 盈利能力指标

| 指标 | 公式 | 判断标准 |
|------|------|----------|
| 毛利率 | (营收-成本)/营收 | 稳定或上升 |
| 净利率 | 净利润/营收 | >10%良好，>20%优秀 |
| ROE | 净利润/净资产 | >15%良好，>20%优秀 |
| ROA | 净利润/总资产 | >5%良好，>10%优秀 |
| ROIC | 税后经营利润/投入资本 | > WACC = 创造价值 |

**ROE驱动分解**：
```
ROE = 净利率 × 资产周转率 × 权益乘数
     ↓         ↓           ↓
   产品盈利   运营效率    财务杠杆
```

### 2. 偿债能力指标

| 指标 | 公式 | 判断标准 |
|------|------|----------|
| 流动比率 | 流动资产/流动负债 | >2很强，1.5-2健康，<1危险 |
| 速动比率 | (流动资产-存货)/流动负债 | >1健康 |
| 资产负债率 | 总负债/总资产 | <60%健康，>70%高风险 |
| 利息保障倍数 | EBIT/利息 | >5很强，<1.5危险 |

### 3. 运营效率指标

| 指标 | 公式 | 判断标准 |
|------|------|----------|
| 存货周转天数 | 365/(成本/平均存货) | <60天良好，>90天需关注 |
| 应收周转天数 | 365/(营收/平均应收) | <60天良好，>90天回款慢 |
| 总资产周转率 | 营收/平均总资产 | 制造业0.5-1.5次正常 |

### 4. 成长性指标

| 指标 | 判断标准 |
|------|----------|
| 营收增长率 | >30%高速，15-30%快速，<5%缓慢 |
| 净利润增长率 | 健康特征：利润增速 ≥ 营收增速 |

## 四、红旗信号识别

### 资产负债表红旗
- 🔴 应收账款增速 > 营收增速（收入确认激进）
- 🔴 存货持续积压（滞销风险）
- 🔴 商誉占比 > 20%（减值风险高）
- 🔴 资产负债率持续上升

### 利润表红旗
- 🔴 毛利率异常波动
- 🔴 投资收益占比过高
- 🔴 营业外收入大增
- 🔴 经营现金流 < 净利润

### 现金流红旗
- 🔴 经营现金流持续为负
- 🔴 自由现金流持续为负
- 🔴 靠融资+卖资产维持

### 审计红旗
- 🔴 审计报告非标
- 🔴 频繁更换审计机构
- 🔴 关联交易异常

## 五、行业研究方法论

### 1. 行业生命周期判断

| 阶段 | 渗透率 | 增速特征 | 投资策略 |
|------|--------|----------|----------|
| 导入期 | <10% | 不稳定 | 高风险高收益 |
| 成长期 | 10-50% | >20% | ✅ 最佳投资期 |
| 成熟期 | 50-80% | 5-15% | 收益稳定 |
| 衰退期 | >80% | <0% | 回避 |

### 2. 波特五力模型

```
┌─────────────────────────────────────────────────────────────┐
│                       行业竞争格局                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 现有竞争者                                              │
│     ├─ 集中度：CR3/CR5                                      │
│     ├─ 差异化：产品/品牌/渠道                               │
│     └─ 增长率：行业增速                                    │
│                                                             │
│  2. 潜在进入者                                              │
│     ├─ 进入壁垒：资金/技术/渠道                             │
│     └─ 威胁程度：高/中/低                                  │
│                                                             │
│  3. 替代品威胁                                              │
│     ├─ 替代品性价比                                         │
│     └─ 转换成本                                            │
│                                                             │
│  4. 供应商议价                                              │
│     ├─ 供应商集中度                                         │
│     └─ 原材料替代性                                         │
│                                                             │
│  5. 购买者议价                                              │
│     ├─ 客户集中度                                           │
│     └─ 转换成本                                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3. 商业模式评估

**好模式特征**：
- ✅ 可复制、高粘性
- ✅ 高毛利、现金流好
- ✅ 轻资产、有护城河

**护城河类型**：
| 类型 | 代表公司 | 特点 |
|------|----------|------|
| 品牌护城河 | 茅台 | 高品牌溢价 |
| 网络效应 | 腾讯 | 用户越多价值越大 |
| 转换成本 | 苹果 | 生态系统锁定 |
| 成本优势 | 拼多多 | 规模效应 |

## 六、估值方法论

### 1. 相对估值法

#### PE（市盈率）
```
PE = 股价/每股收益

判断标准：
├─ <15倍：低估
├─ 15-25倍：合理
└─ >40倍：高估

适用：盈利稳定公司
陷阱：周期股顶部低PE
```

#### PEG
```
PEG = PE/盈利增长率

判断标准：
├─ <1：低估
├─ =1：合理
└─ >2：高估

适用：成长股
```

#### PB（市净率）
```
PB = 股价/每股净资产

判断标准：
├─ <1：破净
├─ 1-2：合理
└─ >3：高估

适用：重资产、银行
注意：要结合ROE看
```

#### PS（市销率）
```
PS = 市值/营收

适用：亏损公司、高增长
优势：销售额难操纵
```

#### EV/EBITDA
```
EV/EBITDA = 企业价值/息税折旧摊销前利润

适用：跨公司比较、并购
优势：不受资本结构影响
```

### 2. 绝对估值法

#### DCF（现金流折现）
```
企业价值 = Σ 未来自由现金流折现

关键参数：
├─ 折现率（WACC）：资金成本+风险溢价
├─ 永续增长率：通常2-3%
└─ 预测期：5-10年

简化公式：
合理PE ≈ 1/(要求回报率 - 永续增长率)
```

**注意事项**：
- 估值是区间不是精确数字
- 做敏感性分析
- 结合相对估值
- 留安全边际

### 3. 估值方法选择

| 行业 | 首选方法 | 备选方法 |
|------|----------|----------|
| 消费/医药 | PE、PEG | DCF |
| 银行/保险 | PB | - |
| 科技/SaaS | PS、PEG | - |
| 周期股 | PB（底部） | - |
| 制造业 | EV/EBITDA | PE |
| 公用事业 | DCF | PE |

### 4. 估值陷阱

| 陷阱 | 说明 | 对策 |
|------|------|------|
| 周期股顶部低PE | 盈利高点，PE看似低 | 用PB判断 |
| 成长股高PE | 高增长支撑高估值 | 看PEG |
| 亏损公司 | PE无意义 | 用PS、PB |
| 一次性收益 | EPS虚高 | 扣非PE |

## 七、风险控制方法论

### 1. 风险识别

| 风险类型 | 检测指标 | 预警阈值 |
|----------|----------|----------|
| 财务风险 | 资产负债率 | >70% |
| 经营风险 | 毛利率变化 | 单季降>5% |
| 现金流风险 | 净现比 | <0.5 |
| 估值风险 | PEG | >2 |

### 2. 止损方法论

**ATR止损**：
```
止损价 = 买入价 - N × ATR(14)

N值选择：
├─ 极低风险：2倍ATR
├─ 低风险：2.5倍ATR
├─ 中等风险：3倍ATR
├─ 高风险：3.5倍ATR
└─ 极高风险：4倍ATR
```

**固定百分比止损**：
- 硬止损：-15%（最后防线）

### 3. 仓位管理

**单股仓位上限**：
| 风险等级 | 仓位上限 |
|----------|----------|
| 极低风险 | 15% |
| 低风险 | 12% |
| 中等风险 | 8% |
| 高风险 | 3% |
| 极高风险 | 0% |

**Beta调整**：
```
调整后仓位 = 基础仓位 × (1/Beta)
```

## 八、分析方法论要点速查

### 财务分析三步法
1. 看ROE → 盈利能力
2. 看现金流 → 利润质量
3. 看负债率 → 财务风险

### 估值判断三问
1. PE是否合理？（看历史分位）
2. PEG是否<1？（看增长支撑）
3. 安全边际够不够？（留20%以上）

### 行业研究三要素
1. 生命周期阶段 → 成长期最佳
2. 竞争格局 → 龙头优势
3. 护城河 → 可持续竞争力

## 十、评分阈值与仓位建议

### 综合评分模型（六维度）

| 维度 | 权重 | 数据来源 | 说明 |
|------|------|----------|------|
| 基本面 | 25% | stock_finance | 财务健康度 |
| 行业前景 | 20% | stock_concept_strength | 概念强度排名 |
| 估值 | 25% | stock_kline + stock_finance | PE/PB/PEG |
| 技术面 | 15% | stock_kline + stock_fund_flow | K线+资金流向 |
| 市场环境 | 10% | global_index + dragon_tiger | 全球市场+机构行为 |
| 短线信号 | 5% | 分时+盘口+技术指标 | 短线买卖信号 |

### 评分评级标准

| 分数 | 评级 | 建议 | 建议仓位 |
|------|------|------|----------|
| ≥80 | 强烈买入 ⭐⭐⭐⭐⭐ | 积极布局 | 12-15% |
| 70-79 | 买入 ⭐⭐⭐⭐ | 逢低布局 | 8-12% |
| 60-69 | 持有 ⭐⭐⭐ | 维持仓位 | 维持现有 |
| 50-59 | 观望 ⭐⭐ | 考虑减仓 | 考虑减仓 |
| <50 | 卖出 ⭐ | 建议清仓 | 0% |

### 调仓规则

**加仓条件（需同时满足）**：
- ✅ 评分 ≥ 70
- ✅ 现价 < 成本价（摊低成本）
- ✅ 仓位 < 建议仓位
- ✅ 现金充足

**减仓条件（满足任一）**：
- ⚠️ 评分 < 60
- ⚠️ 仓位 > 建议仓位
- ⚠️ 到达目标价

**清仓条件（满足任一）**：
- 🔴 评分 < 50
- 🔴 跌破止损位
- 🔴 基本面恶化
