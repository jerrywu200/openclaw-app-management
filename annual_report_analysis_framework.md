### A1. 营收/利润增速变化

```python
# 获取财务数据
finance = get_stock_finance('300308', limit=5)  # 获取5年数据

# 年度营收增速
revenue_growth = finance['营业总收入同比增长'].tolist()

# 年度净利润增速
profit_growth = finance['归属净利润同比增长'].tolist()

# 判断趋势
if revenue_growth[-1] > revenue_growth[-2]:
    revenue_trend = "加速增长 🟢"
elif revenue_growth[-1] > 0:
    revenue_trend = "稳定增长"
elif revenue_growth[-1] > revenue_growth[-2]:
    revenue_trend = "增速回升 🟡"
else:
    revenue_trend = "增速下滑 🔴"

# 利润增速 vs 营收增速
if profit_growth[-1] > revenue_growth[-1]:
    profit_quality = "利润增速>营收增速 → 经营效率提升 🟢"
elif profit_growth[-1] < revenue_growth[-1] * 0.8:
    profit_quality = "利润增速远低于营收 → 成本压力增大 🔴"
else:
    profit_quality = "利润增速与营收基本匹配"
```

## Part B: 🆕 业务定性分析（MD&A解读）

> **核心理念**: 数字是结果，业务是原因。年报中最有价值的部分是管理层讨论与分析（MD&A）。

### B1. 核心战略解读

```
【分析维度】

1. 战略方向变化
   - 公司是否调整了战略方向？
   - 新战略的可行性和执行情况
   - 战略调整的行业背景

2. 战略执行进度
   - 关键战略项目的完成情况
   - 战略里程碑达成情况
   - 战略执行的风险和挑战

3. 战略与财务匹配度
   - 资本开支是否支撑战略方向
   - 研发投入是否匹配战略目标
   - 战略执行是否带来财务回报

【判断标准】

✅ 好的战略：清晰、可执行、有差异化、有护城河
🔴 差的战略：模糊、口号化、随行业随波逐流
```

### B2. 市占率变化

```
【数据来源】
- 年报MD&A中的市场份额披露
- 行业协会数据
- 研报中的市占率分析
- Tavily搜索行业排名

【分析要点】

1. 市占率绝对值
   - 行业龙头？行业前三？细分领域龙头？
   - 市占率是否达到垄断临界点（30%+）

2. 市占率变化趋势
   - 市占率上升 → 竞争力增强 🟢
   - 市占率下降 → 竞争压力增大 🔴
   - 市占率稳定 → 行业格局固化

3. 市占率变化原因
   - 产品竞争力提升？
   - 行业整合并购？
   - 竞争对手退出？
   - 行业整体增长？

【案例】
中际联合：年报披露风电高空作业设备市占率约70%，寡头垄断地位
洛阳钼业：全球钴市占率约15%，全球前三大钴生产商
```

### B3. 新产品/新产能进度

```
【分析维度】

1. 新产品
   - 年报披露的新产品类型
   - 新产品对营收的贡献预期
   - 新产品的市场竞争力
   - 新产品研发进度

2. 新产能
   - 年报披露的产能扩张计划
   - 产能扩张的资金来源
   - 产能扩张的预期回报
   - 产能扩张的风险

3. 产能利用率
   - 当前产能利用率
   - 产能瓶颈是否解决
   - 产能过剩风险

【判断标准】

✅ 好的产能扩张：产能利用率高 + 市场需求旺盛 + 资金充裕
🔴 差的产能扩张：产能利用率低 + 行业产能过剩 + 高负债扩张

【案例】
大金重工：2025年产能扩张至100万吨，对应塔筒需求旺盛
中际旭创：800G光模块产能快速扩张，匹配AI算力需求爆发
```

### B4. 管理层质量评估

```
【评估维度】

1. 战略眼光
   - 是否提前布局行业趋势
   - 是否有前瞻性判断
   - 是否抓住行业机会

2. 执行能力
   - 战略执行是否到位
   - 目标达成率
   - 危机应对能力

3. 沟通透明度
   - 年报披露是否详细透明
   - 是否诚实面对挑战
   - 是否给出明确指引

4. 股东回报意识
   - 分红政策是否稳定
   - 是否重视股东回报
   - 资本配置是否合理

【红旗信号】

🔴 频繁更换战略方向 → 战略摇摆
🔴 目标总是未达成 → 执行能力差
🔴 年报避重就轻 → 沟通不透明
🔴 长期不分红或过度分红 → 资本配置不合理
```

### B5. 行业竞争格局变化

```
【分析维度】

1. 行业集中度变化
   - 行业是否在整合
   - 龙头企业份额提升还是下降
   - 新进入者威胁

2. 竞争对手动态
   - 竞争对手的产能扩张
   - 竞争对手的产品升级
   - 竞争对手的价格策略

3. 行业价格变化
   - 行业是否出现价格战
   - 价格战的原因和影响
   - 价格战是否接近尾声

【数据来源】
- 年报MD&A中的行业分析
- 研报中的行业格局分析
- Tavily搜索行业动态

【判断标准】

✅ 行业格局改善：龙头份额提升 + 行业集中度提高 + 价格稳定
🔴 行业格局恶化：价格战 + 新进入者 + 龙头份额下降
```

## Part C: 🆕 现金流与资本配置分析

> **核心理念**: 在当前宏观环境下，市场越来越看重"股东回报"。自由现金流（FCF）的创造能力，以及公司赚了钱之后怎么花，是决定估值的核心。

### C1. 自由现金流（FCF）创造能力

```python
# FCF计算公式
# FCF = 经营现金流 - 资本开支

# 从财务数据获取
operating_cash_flow = finance['经营活动产生的现金流量净额'].iloc[0]
capex = finance['购建固定资产、无形资产和其他长期资产支付的现金'].iloc[0]

fcf = operating_cash_flow - capex

# FCF趋势判断
fcf_history = []
for i in range(len(finance)):
    ocf = finance.iloc[i]['经营活动产生的现金流量净额']
    cap = finance.iloc[i]['购建固定资产、无形资产和其他长期资产支付的现金']
    fcf_history.append(ocf - cap)

if fcf > 0 and fcf_history[-1] > fcf_history[-2]:
    fcf_trend = "FCF为正且增长 → 现金创造能力强 🟢"
elif fcf > 0:
    fcf_trend = "FCF为正 → 有现金创造能力"
elif fcf < 0 and operating_cash_flow > 0:
    fcf_trend = "FCF为负但经营现金流为正 → 高投入期 🟡"
else:
    fcf_trend = "FCF为负且经营现金流为负 → 现金危机 🔴"

# FCF/净利润比率
fcf_to_profit = fcf / finance['归属净利润'].iloc[0]
if fcf_to_profit > 0.8:
    profit_quality = "利润质量高（FCF/净利润>0.8） 🟢"
elif fcf_to_profit > 0.5:
    profit_quality = "利润质量尚可"
else:
    profit_quality = "利润质量低（账面利润≠真金白银） 🔴"
```

### C2. 资本开支（Capex）趋势

```python
# Capex趋势
capex_history = finance['购建固定资产、无形资产和其他长期资产支付的现金'].tolist()

# Capex/营收比率
capex_to_revenue = capex_history[-1] / finance['营业总收入'].iloc[0]

# Capex趋势判断
if capex_history[-1] > capex_history[-2] * 1.5:
    capex_trend = "Capex快速增长 → 高投入扩张期 🟡"
elif capex_history[-1] < capex_history[-2] * 0.5:
    capex_trend = "Capex大幅收缩 → 收缩期或保守期 🔴"
else:
    capex_trend = "Capex稳定"

# Capex回报率判断
# ROIC = 税后经营利润 / 投入资本
# 如果ROIC > WACC，Capex在创造价值
# 如果ROIC < WACC，Capex在毁灭价值
```

### C3. 分红政策变化

```
【分析维度】

1. 分红率变化
   - 年报披露的分红方案
   - 分红率（分红/净利润）历史对比
   - 分红率变化原因

2. 分红稳定性
   - 分红是否连续多年
   - 分红金额是否稳定增长
   - 分红政策是否透明

3. 分红与成长性平衡
   - 高成长公司低分红合理
   - 低成长公司应提高分红
   - 分红不应影响核心业务投资

【判断标准】

✅ 健康分红：分红率30-50% + 分红连续稳定 + 有足够留存支持发展
🔴 不健康分红：分红率>80%透支未来 或 分红率<10%长期不回馈股东

【A股常见分红率区间】

- 高成长科技股：10-20%（留存用于扩张）
- 稳定消费股：30-50%（平衡增长与回报）
- 公用事业股：50-70%（现金流稳定，低增长）
- 银行股：30%左右（监管要求）
```

### C4. 股东回报（回购/分红率）

```
【分析维度】

1. 股东回报率（Total Shareholder Return）
   TSR = 分红收益率 + 股价增长率

2. 回购政策
   - 是否有股份回购计划
   - 回购金额和进度
   - 回购对每股收益的影响

3. 股东回报对比
   - 与同行业对比
   - 与无风险利率对比
   - 与市场平均对比

【案例】
洛阳钼业：2024年分红率约30%，稳定回馈股东
中国卫通：分红率较低，主要留存用于卫星网络建设
```

### C5. 资本配置效率评估

```
【评估框架】

1. 资本配置去向
   - 资本开支（产能扩张）
   - 研发投入（技术创新）
   - 分红（股东回报）
   - 储备现金（安全边际）
   - 债务偿还（降低风险）

2. 资本配置效率
   - ROIC（投入资本回报率）
   - ROIC > WACC = 创造价值
   - ROIC < WACC = 灭绝价值

3. 资本配置历史表现
   - 过去投资项目的回报率
   - 过去收购项目的整合效果
   - 过去研发投入的产品化率

【判断标准】

✅ 高效配置：ROIC持续>WACC + 增长投入有回报 + 分红稳定
🔴 低效配置：ROIC<WACC + 乱投资 + 长期不分红

【公式】

ROIC = 税后经营利润 / (股权 + 有息负债 - 现金)
WACC = 股权成本 × 股权比例 + 债务成本 × 债务比例
```

## Part D: 🆕 前瞻性指引分析

> **核心理念**: 年报不仅是总结过去，更重要的是管理层对下一年的业绩指引。很多时候，过去业绩符合预期，但因为指引偏弱，股价依然大跌。

### D1. 管理层业绩指引

```
【数据来源】
- 年报MD&A中的业绩指引
- 年报披露后的投资者交流会
- 研报中的业绩预测（作为市场预期基准）

【指引类型】

1. 定量指引
   - 营收增长目标
   - 利润增长目标
   - 产能扩张目标
   - 市占率目标

2. 定性指引
   - 行业趋势判断
   - 战略重点方向
   - 风险因素提示

【分析要点】

1. 指引的可信度
   - 过往指引达成率
   - 指引是否保守或激进
   - 指引的行业背景合理性

2. 指引与市场预期对比
   - 指引高于市场预期 → 股价可能上涨 🟢
```

### F2. 估值倍数（PE/PB）调整

```
【估值倍数调整原因】

1. 盈利增长预期变化
   - 增长预期上调 → PE倍数可上调
   - 增长预期下调 → PE倍数应下调

2. 行业估值水平变化
   - 行业整体估值上升 → PE可上调
   - 行业整体估值下降 → PE应下调

3. 公司风险变化
   - 风险降低（财务稳健） → PE可上调
   - 风险上升（财务恶化） → PE应下调

4. 市场情绪变化
   - 市场乐观 → PE可上调
   - 市场悲观 → PE应下调

【公式】

合理PE ≈ (预期增长率 + 分红率) / (要求回报率 - 永续增长率)

PEG = PE / 增长率
PEG < 1 估值偏低
PEG = 1 估值合理
PEG > 2 估值偏高
```

### F3. 估值调整原因分解

```
【分解框架】

目标价调整 = EPS变化 × 估值倍数变化

分解为：

1. EPS变化原因
   ├─ 营收增长预测调整（贡献XX%）
   ├─ 毛利率预测调整（贡献XX%）
   ├─ 费用率预测调整（贡献XX%）
   └─ 其他因素（贡献XX%）

2. 估值倍数变化原因
   ├─ 增长预期变化（贡献XX%）
   ├─ 行业估值变化（贡献XX%）
   ├─ 公司风险变化（贡献XX%）
   └─ 市场情绪变化（贡献XX%）

【案例】

某公司原目标价100元（EPS 5元 × PE 20倍）

年报后调整：
- EPS下调至4元（-20%）：毛利率下降导致
- PE下调至15倍（-25%）：增长预期下调导致
- 新目标价 = 4 × 15 = 60元（-40%）

分解说明：
- EPS下调贡献-20%
- PE下调贡献-25%
- 总调整-40%（非线性叠加）

【报告格式】

目标价调整原因分解：

| 因素 | 原预测 | 新预测 | 变化 | 贡献度 |
|------|--------|--------|------|--------|
| EPS | 5元 | 4元 | -20% | XX% |
| PE倍数 | 20倍 | 15倍 | -25% | XX% |
| 目标价 | 100元 | 60元 | -40% | 100% |

EPS下调原因：
- 毛利率下降2个百分点 → EPS下调0.5元
- 营收增速下调5个百分点 → EPS下调0.3元
- 其他因素 → EPS下调0.2元

PE下调原因：
- 增长预期从25%下调至15% → PEG从0.8升至1.0 → PE下调
- 行业估值中枢下移 → PE下调
- 公司财务风险上升 → PE下调
```

### F4. 目标价重新计算

```python
# 目标价计算方法

# 方法1：PE法
new_target_price = new_eps * new_pe

# 方法2：PEG法
peg_target_pe = expected_growth_rate  # PEG=1时的合理PE
peg_target_price = new_eps * peg_target_pe

# 方法3：DCF法（简化）

# 方法4：历史估值法
historical_pe_avg = historical_pe.mean()
historical_target = new_eps * historical_pe_avg

# 综合目标价（加权平均）
final_target = (pe_target * 0.4 + peg_target * 0.3 + dcf_target * 0.2 + historical_target * 0.1)
```

### F5. 风险溢价变化

```
【风险溢价因素】

1. 公司特有风险
   - 财务风险（负债率、现金流）
   - 经营风险（竞争格局、技术迭代）

   - 利率风险
   - 汇率风险

【风险溢价调整】

风险溢价上升 → 要求回报率上升 → 合理估值下降
风险溢价下降 → 要求回报率下降 → 合理估值上升

【公式】

要求回报率 = 无风险利率 + 市场风险溢价 + 公司风险溢价

低风险公司：要求回报率8-10%
```

### 标识格式

在年报分析报告中，对于缺失或需要人工提供的数据，使用以下格式标识：

```
【缺失数据】⚠️
- 数据项：XXX
- 缺失原因：自动获取失败 / 数据库无此字段 / 需人工提取
- 替代方案：XXX（如有）
- 需要提供：请提供XXX数据

【需人工提供】👤
- 数据项：XXX
- 说明：该数据无法自动获取，需要从年报原文提取
- 提供方式：请阅读年报第XX页，提供XXX信息
```

### 标识分类

| 标识 | 含义 | 示例 |
|------|------|------|
| `【缺失数据】⚠️` | 自动获取失败 | 财务数据缺失、API限流 |
| `【需人工提供】👤` | 需要手工提取 | MD&A内容、战略规划 |
| `【数据不确定】❓` | 数据可能不准确 | 估算值、预测值 |
