# 股票分析 Skills 配置

> 安装时间：2026年3月24日

---

## 已安装的 Skills

### 1. china-stock-analysis
**功能**：中国股票价格分析和投资建议

**使用场景**：
- 查询A股、港股价格
- 技术面分析
- 提供买入/持有/卖出建议

**触发方式**：
- "分析贵州茅台的股价"
- "比亚迪现在可以买吗"
- "分析 0700.HK 腾讯"

---

### 2. stock-valuation-monitor
**功能**：股票和ETF估值监控

**核心能力**：
- PE/PB历史百分位计算
- 估值区间评估（机会/合理/风险）
- ETF折溢价分析
- 批量查询支持

**估值区间定义**：
| 区间 | PE百分位 | 建议 |
|------|----------|------|
| 机会区间 | < 30% | 积极关注，适合定投 |
| 合理区间 | 30%-70% | 持有观望 |
| 风险区间 | > 70% | 考虑减仓 |

**触发方式**：
- "查询 300327 的估值"
- "评估沪深300ETF的投资价值"
- "监控 600519 的估值区间"

---

## 依赖项

- Python 3.8+
- akshare（A股数据源）
- pandas, numpy
- requests

## 使用示例

### 示例1：单只股票估值查询
```
查询中际旭创（300308）的估值情况
```

### 示例2：多只股票比较
```
比较中际旭创、新易盛、华工科技的估值水平
```

### 示例3：行业龙头分析
```
分析英伟达产业链核心公司的估值：中际旭创、沪电股份、工业富联
```

## 注意事项

1. 数据仅供参考，不构成投资建议
2. 估值分析基于历史数据
3. 新股历史数据可能不足
4. 建议结合基本面分析
