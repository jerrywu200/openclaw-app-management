## 一、行业生命周期判断

| 阶段 | 渗透率 | 增速 | 特征 | 投资策略 |
|------|--------|------|------|----------|
| 导入期 | <10% | 不稳定 | 技术不成熟，商业模式探索 | 风险投资，高赔率 |
| **成长期** | 10-50% | >20% | 快速增长，格局未定 | **最佳投资期** ⭐ |
| 成熟期 | 50-80% | 5-15% | 格局稳定，龙头盈利 | 价值投资，分红 |
| 衰退期 | >80% | <0% | 需求萎缩，产能过剩 | 回避或做空 |

## 三、产业链分析方法

### 3.1 产业链图谱绘制

```
【上游】 → 【中游】 → 【下游】

原材料 → 制造加工 → 终端应用
   │          │          │
   ↓          ↓          ↓
 供应商    生产商      客户
```

### 3.2 价值分布分析

| 环节 | 分析维度 | 关键问题 |
|------|----------|----------|
| 上游 | 利润占比、技术壁垒 | 谁掌握核心资源/技术？ |
| 中游 | 利润占比、竞争强度 | 哪个环节最有价值？ |
| 下游 | 利润占比、议价能力 | A股公司在哪个环节？ |

### 3.3 A股投资机会识别

```
判断标准：
├─ 价值占比高（利润集中环节）
├─ A股公司有竞争力
├─ 国产替代空间大
└─ 技术壁垒可突破
```

## 五、投资逻辑提炼

### 5.1 驱动因素分析

```
核心驱动因素识别：

1. 量价拆解
   ├─ 销量驱动：渗透率提升、市场扩张
   └─ 价格驱动：供需关系、成本变化

2. 政策驱动
   ├─ 补贴政策
   ├─ 强制标配
   └─ 出口鼓励

3. 技术驱动
   ├─ 技术突破
   ├─ 成本下降
   └─ 新应用场景

4. 周期驱动
   ├─ 库存周期
   ├─ 产能周期
   └─ 价格周期
```

### 5.2 关键跟踪指标

按行业类型区分：

| 行业类型 | 关键跟踪指标 |
|----------|--------------|
| 制造业 | 产能利用率、库存周转、订单增速 |
| 周期品 | 价格指数、库存水平、开工率 |
| 消费品 | 渠道库存、终端动销、品牌声量 |
| 科技品 | 技术迭代、专利数量、研发投入 |
| 金融业 | 利率环境、资产质量、监管政策 |
