# Alpha - AI 股票分析助手

基于 OpenClaw 构建的 AI 股票分析助手，专注于 A 股投资分析。

## 功能特性

- 📊 V3.5 个股分析框架（十步法）
- 📈 六维度综合评分模型
- ⚠️ 五维风险分解体系
- 🔄 持续跟踪与预警机制

## 核心文件

| 文件 | 说明 |
|------|------|
| `stock_analysis_framework.md` | V3.5 十步法分析框架 |
| `stock_scoring_model.py` | 六维度综合评分模型 |
| `tracking_monitor.py` | 持续跟踪与预警 |
| `MEMORY.md` | 长期记忆 |

## 目录结构

```
├── *.md                    # 核心配置和分析框架
├── *.py                    # 工具脚本
├── learning/              # 学习笔记
└── memory/                # 记忆文件
```
