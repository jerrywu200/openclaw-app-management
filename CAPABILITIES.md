### 🎉 V3.5 新增能力（2026-03-30）

#### 1. 财务趋势分析（Phase 1）
- **模块**: `finance_data_filter.py`
- **功能**: 多期财务指标对比，判断改善/恶化
- **评分**: 基本面评分 = 当期评分 + 趋势评分（-19 ~ +19分）
- **使用**:
  ```python
  from finance_data_filter import analyze_financial_trend
  trend = analyze_financial_trend(finance_df, years=3)
  ```

#### 2. 评分置信度机制（Phase 2）
- **模块**: `stock_scoring_model.py`
- **功能**: 三维度评估评分可靠性
- **公式**: 置信度 = 完整度×40% + 时效性×30% + 一致性×30%
- **等级**: 高/中高/中/中低/低

#### 3. 同行估值对比（Phase 3）
- **模块**: `stock_scoring_model.py`
- **功能**: PE/PB相对位置评分，PB-ROE匹配度
- **评分**: PE相对评分(20分) + PB-ROE匹配评分(10分)

#### 4. 五维风险分解（Phase 4）
- **模块**: `stock_scoring_model.py`
- **功能**: 五维度风险量化
- **维度**: 财务(25%) + 估值(25%) + 行业(20%) + 技术(15%) + 流动性(15%)
- **等级**: 低风险(≤15%) / 中低风险(≤12%) / 中等风险(≤10%) / 中高风险(≤5%) / 高风险(0%)

#### 5. 持续跟踪机制（Phase 5）
- **模块**: `tracking_monitor.py`
- **功能**: 评分历史记录 + 预警检测 + 心跳集成
- **预警级别**: 🔴critical / 🟡warning / 🟢info
- **使用**:
  ```python
  from tracking_monitor import record_score_history, check_alerts
  record_score_history(stock_code, stock_name, analysis_result)
  ```

## 四、风险防御能力

### 框架文件
- **主框架**: `risk_defense_framework.md` V2.0（双频段架构）

### 核心能力

#### 基本面风控（低频，季度更新）
| 层级 | 风险类型 | 核心指标 |
|------|----------|----------|
| 第一层 | 财务风险 | 偿债/现金流/利润质量/资产质量 |
| 第二层 | 经营风险 | 行业周期/竞争格局/客户集中度 |
| 第三层 | 管理层风险 | 战略执行/公司治理 |

#### 交易面风控（高频，每日更新）
| 层级 | 风险类型 | 核心指标 |
|------|----------|----------|
| 第四层 | 估值风险 | PE/PB/PEG分位 |
| 第五层 | 技术风险 | 趋势/支撑压力/量价配合 |
| 第六层 | 情绪风险 | 资金流向/龙虎榜/舆情 |

### 输出成果
- 风险评分（0-50分）
- 风险等级（极低/低/中/高/极高）
- 仓位上限（0-15%）
- ATR止损位

### 特色功能
- 动态权重自适应（按行业/生命周期调整）
- 双频段架构（基本面+交易面分离）
- ATR波动率止损（替代固定百分比）
